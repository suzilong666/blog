# 变量与数据类型

[官方文档](https://docs.python.org/zh-cn/3/tutorial/introduction.html#using-python-as-a-calculator)

变量和数据类型是编程的基础。Python 是一种动态类型语言，变量不需要声明类型，解释器会根据赋值自动推断。本文将介绍 Python 的变量命名规则和主要数据类型。

## 变量命名规则

### 合法变量名

```python
# 基本规则：字母、数字、下划线，不能以数字开头
name = "张三"
age = 25
user_name = "李四"
_private = "私有变量"
PI = 3.14159  # 常量通常用大写
```

### 命名规范

- **小写+下划线**：普通变量（`user_name`）
- **大写+下划线**：常量（`MAX_SIZE`）
- **驼峰命名**：类名（`MyClass`）
- **单/双下划线前缀**：特殊含义（`_private`、`__dunder__`）

### 非法变量名

```python
# 2name = "错误"      # 不能以数字开头
# class = "错误"      # 不能使用关键字
# user-name = "错误"  # 不能使用连字符
```

> 💡 **提示**：使用有意义的变量名，避免 `a`、`b`、`temp` 等模糊命名。

## 基本数据类型

### 整数（int）

表示正整数、负整数和零。

```python
# 整数示例
positive = 42
negative = -17
zero = 0

# 不同进制
decimal = 100      # 十进制
binary = 0b1100100 # 二进制（等于 100）
octal = 0o144      # 八进制（等于 100）
hexadecimal = 0x64 # 十六进制（等于 100）

print(type(positive))  # <class 'int'>
```

Python 3 的整数没有大小限制，可以处理任意大的数：

```python
big_number = 2 ** 1000  # 非常大的数，Python 也能处理
```

### 浮点数（float）

表示带小数点的数字。

```python
# 浮点数示例
pi = 3.14159
temperature = -5.5
scientific = 1.23e-4  # 科学计数法（等于 0.000123）

print(type(pi))  # <class 'float'>
```

> ⚠️ **注意**：浮点数运算可能存在精度问题：
```python
print(0.1 + 0.2)  # 0.30000000000000004（不是精确的 0.3）
```

需要精确计算时使用 `decimal` 模块。

### 布尔值（bool）

表示真（`True`）或假（`False`）。

```python
is_active = True
is_deleted = False

print(type(is_active))  # <class 'bool'>

# 布尔运算
print(True and False)   # False
print(True or False)    # True
print(not True)         # False
```

### None 类型

表示空值或无值。

```python
result = None

print(type(result))  # <class 'NoneType'>

# 检查是否为 None
if result is None:
    print("结果为空")
```

> 💡 **最佳实践**：使用 `is None` 而不是 `== None` 进行检查。

## 类型转换

### 隐式转换

Python 会自动进行某些类型转换：

```python
# 整数和浮点数运算，结果为浮点数
result = 5 + 2.5
print(result)       # 7.5
print(type(result)) # <class 'float'>

# 布尔值和整数
print(True + 1)     # 2（True 被视为 1）
print(False * 5)    # 0（False 被视为 0）
```

### 显式转换

使用内置函数进行类型转换：

```python
# 转换为整数
print(int(3.14))      # 3（截断小数部分）
print(int("42"))      # 42
# print(int("3.14"))  # ValueError!

# 转换为浮点数
print(float(5))       # 5.0
print(float("3.14"))  # 3.14

# 转换为字符串
print(str(42))        # "42"
print(str(3.14))      # "3.14"

# 转换为布尔值
print(bool(0))        # False
print(bool(1))        # True
print(bool(""))       # False（空字符串）
print(bool("hello"))  # True
```

### 常见转换错误

```python
# int("3.14")     # ValueError: 无法将包含小数的字符串直接转为整数
# float("abc")    # ValueError: 无法将非数字字符串转为浮点数

# 正确做法
print(int(float("3.14")))  # 3（先转浮点，再转整数）
```

## 动态类型特性

Python 是动态类型语言，变量可以在运行时改变类型：

```python
value = 42        # 整数
print(type(value))  # <class 'int'>

value = "Hello"   # 现在是字符串
print(type(value))  # <class 'str'>

value = 3.14      # 现在是浮点数
print(type(value))  # <class 'float'>
```

> ⚠️ **注意**：虽然灵活，但频繁改变变量类型会降低代码可读性，应谨慎使用。

## 查看类型

使用 `type()` 函数查看变量的类型：

```python
x = 42
print(type(x))  # <class 'int'>

y = 3.14
print(type(y))  # <class 'float'>

# 使用 isinstance() 检查类型
print(isinstance(x, int))    # True
print(isinstance(y, float))  # True
print(isinstance(x, float))  # False
```

> 💡 **最佳实践**：优先使用 `isinstance()` 而不是 `type()` 进行类型检查，因为它支持继承关系。

## 常见问题

### 1. 变量未定义错误

```python
# print(undefined_var)  # NameError: name 'undefined_var' is not defined

# 确保先赋值再使用
my_var = 10
print(my_var)  # 正常
```

### 2. 类型不匹配错误

```python
# print("年龄：" + 25)  # TypeError: 不能拼接字符串和整数

# 正确做法
print("年龄：" + str(25))  # 年龄：25
print(f"年龄：{25}")       # 年龄：25（推荐）
```

### 3. 整数除法

```python
# Python 3 中 / 总是返回浮点数
print(10 / 3)   # 3.3333333333333335

# 使用 // 进行整数除法
print(10 // 3)  # 3
```
