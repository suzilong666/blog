# 列表与元组

[官方文档](https://docs.python.org/zh-cn/3/tutorial/datastructures.html)

列表和元组是 Python 中最基本的序列数据结构，用于存储有序的元素集合。列表是可变的，而元组是不可变的。本文将详细介绍两者的用法和区别。

## 列表（List）

### 创建列表

```python
# 空列表
empty = []
empty = list()

# 包含元素的列表
fruits = ["apple", "banana", "orange"]
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]

# 从其他可迭代对象创建
text = "Python"
chars = list(text)
print(chars)  # ['P', 'y', 't', 'h', 'o', 'n']
```

### 访问元素

```python
fruits = ["apple", "banana", "orange"]

# 索引访问（从 0 开始）
print(fruits[0])    # apple
print(fruits[1])    # banana
print(fruits[-1])   # orange（最后一个）
print(fruits[-2])   # banana（倒数第二个）

# 切片
print(fruits[0:2])  # ['apple', 'banana']
print(fruits[1:])   # ['banana', 'orange']
print(fruits[:2])   # ['apple', 'banana']
```

### 修改列表

```python
fruits = ["apple", "banana", "orange"]

# 修改元素
fruits[1] = "grape"
print(fruits)  # ['apple', 'grape', 'orange']

# 添加元素
fruits.append("melon")        # 末尾添加
print(fruits)  # ['apple', 'grape', 'orange', 'melon']

fruits.insert(1, "pear")      # 指定位置插入
print(fruits)  # ['apple', 'pear', 'grape', 'orange', 'melon']

# 删除元素
removed = fruits.pop()        # 删除并返回最后一个
print(removed)  # melon
print(fruits)   # ['apple', 'pear', 'grape', 'orange']

fruits.remove("pear")         # 删除指定值
print(fruits)  # ['apple', 'grape', 'orange']

del fruits[0]                 # 删除指定索引
print(fruits)  # ['grape', 'orange']

# 清空列表
fruits.clear()
print(fruits)  # []
```

### 常用方法

```python
numbers = [3, 1, 4, 1, 5, 9, 2, 6]

# 长度
print(len(numbers))  # 8

# 最大值/最小值
print(max(numbers))  # 9
print(min(numbers))  # 1

# 求和
print(sum(numbers))  # 31

# 排序
numbers.sort()
print(numbers)  # [1, 1, 2, 3, 4, 5, 6, 9]

# 降序排序
numbers.sort(reverse=True)
print(numbers)  # [9, 6, 5, 4, 3, 2, 1, 1]

# 反转
numbers.reverse()
print(numbers)  # [1, 1, 2, 3, 4, 5, 6, 9]

# 计数
print(numbers.count(1))  # 2

# 查找索引
print(numbers.index(4))  # 4
```

### 列表推导式

```python
# 基本推导式
squares = [x ** 2 for x in range(1, 6)]
print(squares)  # [1, 4, 9, 16, 25]

# 带条件的推导式
evens = [x for x in range(10) if x % 2 == 0]
print(evens)  # [0, 2, 4, 6, 8]

# 复杂表达式
words = ["hello", "world", "python"]
upper_words = [word.upper() for word in words]
print(upper_words)  # ['HELLO', 'WORLD', 'PYTHON']

# 嵌套推导式
matrix = [[i * j for j in range(1, 4)] for i in range(1, 4)]
print(matrix)
# [[1, 2, 3], [2, 4, 6], [3, 6, 9]]
```

## 元组（Tuple）

### 创建元组

```python
# 空元组
empty = ()
empty = tuple()

# 包含元素的元组
point = (3, 4)
colors = ("red", "green", "blue")

# 单元素元组（需要逗号）
single = (42,)
not_tuple = (42)  # 这是整数！

# 省略括号（可选）
point = 3, 4
print(point)  # (3, 4)
```

### 访问元素

```python
point = (3, 4, 5)

print(point[0])     # 3
print(point[-1])    # 5
print(point[0:2])   # (3, 4)

# 元组不可变，不能修改
# point[0] = 10  # TypeError!
```

### 元组解包

```python
# 基本解包
point = (3, 4)
x, y = point
print(x)  # 3
print(y)  # 4

# 交换变量
a, b = 1, 2
a, b = b, a
print(a, b)  # 2 1

# 星号解包
numbers = (1, 2, 3, 4, 5)
first, *middle, last = numbers
print(first)   # 1
print(middle)  # [2, 3, 4]
print(last)    # 5
```

### 元组方法

```python
numbers = (1, 2, 3, 2, 4, 2)

print(numbers.count(2))  # 3
print(numbers.index(3))  # 2
```

## 列表 vs 元组

| 特性 | 列表 | 元组 |
| :--- | :--- | :--- |
| 语法 | `[]` | `()` |
| 可变性 | 可变 | 不可变 |
| 性能 | 较慢 | 较快 |
| 内存占用 | 较大 | 较小 |
| 适用场景 | 动态数据 | 固定数据 |

### 选择建议

```python
# 使用列表：数据会变化
scores = [90, 85, 92]
scores.append(88)

# 使用元组：数据不变
DAYS_OF_WEEK = ("Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun")
RGB_COLORS = (255, 128, 0)

# 元组作为字典键（因为不可变）
location = {(3, 4): "Point A"}
```

## 常见操作

### 拼接和重复

```python
# 列表拼接
list1 = [1, 2]
list2 = [3, 4]
combined = list1 + list2
print(combined)  # [1, 2, 3, 4]

# 列表重复
repeated = [1, 2] * 3
print(repeated)  # [1, 2, 1, 2, 1, 2]

# 元组同样支持
tuple1 = (1, 2)
tuple2 = (3, 4)
print(tuple1 + tuple2)  # (1, 2, 3, 4)
```

### 成员检查

```python
fruits = ["apple", "banana", "orange"]

print("apple" in fruits)      # True
print("grape" not in fruits)  # True
```

### 遍历

```python
# 直接遍历
fruits = ["apple", "banana", "orange"]
for fruit in fruits:
    print(fruit)

# 带索引遍历
for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")
```

### 复制列表

```python
original = [1, 2, 3]

# 浅拷贝（三种方式等价）
copy1 = original.copy()
copy2 = original[:]
copy3 = list(original)

# 注意：修改副本不影响原列表
copy1.append(4)
print(original)  # [1, 2, 3]
print(copy1)     # [1, 2, 3, 4]
```

## 嵌套结构

### 列表嵌套

```python
# 二维列表（矩阵）
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

print(matrix[0])      # [1, 2, 3]
print(matrix[1][2])   # 6

# 遍历矩阵
for row in matrix:
    for item in row:
        print(item, end=" ")
    print()
```

### 元组嵌套

```python
# 坐标列表
points = [(1, 2), (3, 4), (5, 6)]

for x, y in points:
    print(f"({x}, {y})")
```

## 常见错误

### 1. 可变默认参数

```python
# 危险！
def append_to_list(item, lst=[]):
    lst.append(item)
    return lst

print(append_to_list(1))  # [1]
print(append_to_list(2))  # [1, 2]（不是预期的 [2]）

# 正确做法
def append_to_list(item, lst=None):
    if lst is None:
        lst = []
    lst.append(item)
    return lst
```

### 2. 浅拷贝陷阱

```python
# 浅拷贝：嵌套对象共享引用
original = [[1, 2], [3, 4]]
copy = original.copy()

copy[0].append(3)
print(original)  # [[1, 2, 3], [3, 4]]（原列表也被修改！）

# 深拷贝：完全独立
import copy
original = [[1, 2], [3, 4]]
deep_copy = copy.deepcopy(original)

deep_copy[0].append(3)
print(original)    # [[1, 2], [3, 4]]
print(deep_copy)   # [[1, 2, 3], [3, 4]]
```

### 3. 索引越界

```python
fruits = ["apple", "banana"]
# print(fruits[5])  # IndexError!

# 安全做法
if len(fruits) > 5:
    print(fruits[5])
```

## 实际应用示例

### 1. 栈实现

```python
stack = []

# 入栈
stack.append(1)
stack.append(2)
stack.append(3)

# 出栈
print(stack.pop())  # 3
print(stack.pop())  # 2
print(stack)        # [1]
```

### 2. 队列实现

```python
from collections import deque

queue = deque([1, 2, 3])

# 入队
queue.append(4)

# 出队
print(queue.popleft())  # 1
print(queue.popleft())  # 2
```

### 3. 去重并保持顺序

```python
def unique(items):
    seen = set()
    result = []
    for item in items:
        if item not in seen:
            seen.add(item)
            result.append(item)
    return result

numbers = [3, 1, 4, 1, 5, 9, 2, 6, 5]
print(unique(numbers))  # [3, 1, 4, 5, 9, 2, 6]
```

## 最佳实践

1. **优先使用列表推导式**：简洁高效
2. **不可变数据用元组**：更安全、性能更好
3. **避免深层嵌套**：超过 3 层考虑使用类或字典
4. **使用解包赋值**：提高代码可读性
5. **注意浅拷贝陷阱**：嵌套结构使用深拷贝
