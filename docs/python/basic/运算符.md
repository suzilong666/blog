# 运算符

[官方文档](https://docs.python.org/zh-cn/3/reference/expressions.html#operators)

运算符用于对变量和值执行操作。Python 提供了丰富的运算符，包括算术、比较、逻辑、赋值等多种类型。本文将系统介绍 Python 中的各类运算符及其用法。

## 算术运算符

用于执行数学运算。

```python
a = 10
b = 3

# 加法
print(a + b)   # 13

# 减法
print(a - b)   # 7

# 乘法
print(a * b)   # 30

# 除法（返回浮点数）
print(a / b)   # 3.3333333333333335

# 整除（向下取整）
print(a // b)  # 3

# 取模（余数）
print(a % b)   # 1

# 幂运算
print(a ** b)  # 1000（10 的 3 次方）
```

### 运算符优先级

```python
# 优先级：幂 > 乘除 > 加减
result = 2 + 3 * 4 ** 2
print(result)  # 50（先算 4**2=16，再算 3*16=48，最后 2+48=50）

# 使用括号改变优先级
result = (2 + 3) * 4 ** 2
print(result)  # 80
```

## 比较运算符

用于比较两个值，返回布尔值（True 或 False）。

```python
a = 10
b = 20

# 等于
print(a == b)  # False

# 不等于
print(a != b)  # True

# 大于
print(a > b)   # False

# 小于
print(a < b)   # True

# 大于等于
print(a >= b)  # False

# 小于等于
print(a <= b)  # True
```

### 链式比较

Python 支持优雅的链式比较：

```python
x = 15

# 传统写法
print(x > 10 and x < 20)  # True

# Pythonic 写法
print(10 < x < 20)  # True

# 更多示例
age = 25
print(18 <= age <= 60)  # True（判断是否在范围内）
```

## 逻辑运算符

用于组合多个条件表达式。

```python
# and：两者都为 True 才返回 True
print(True and True)    # True
print(True and False)   # False

# or：任一为 True 就返回 True
print(True or False)    # True
print(False or False)   # False

# not：取反
print(not True)         # False
print(not False)        # True
```

### 实际应用

```python
age = 25
has_license = True

# 判断是否可以开车
if age >= 18 and has_license:
    print("可以开车")

# 判断是否免票（儿童或老人）
child_age = 5
senior_age = 70
if child_age < 6 or senior_age >= 65:
    print("免票")
```

### 短路求值

```python
# and 短路：第一个为 False 时不再评估第二个
def check():
    print("检查执行")
    return True

print(False and check())  # False（check() 不会执行）
print(True and check())   # False（check() 会执行）
```

## 赋值运算符

用于给变量赋值。

```python
# 基本赋值
x = 10

# 加法赋值
x += 5    # 等价于 x = x + 5
print(x)  # 15

# 减法赋值
x -= 3    # 等价于 x = x - 3
print(x)  # 12

# 乘法赋值
x *= 2    # 等价于 x = x * 2
print(x)  # 24

# 除法赋值
x /= 4    # 等价于 x = x / 4
print(x)  # 6.0

# 整除赋值
x //= 2   # 等价于 x = x // 2
print(x)  # 3.0

# 取模赋值
x %= 2    # 等价于 x = x % 2
print(x)  # 1.0

# 幂赋值
x = 2
x **= 10  # 等价于 x = x ** 10
print(x)  # 1024
```

## 成员运算符

用于检查值是否存在于序列（字符串、列表、元组等）中。

```python
# in：检查是否包含
fruits = ["apple", "banana", "orange"]
print("apple" in fruits)     # True
print("grape" in fruits)     # False

# not in：检查是否不包含
print("grape" not in fruits) # True

# 字符串检查
message = "Hello, Python!"
print("Python" in message)   # True
print("Java" in message)     # False
```

## 身份运算符

用于检查两个变量是否指向同一个对象（内存地址）。

```python
a = [1, 2, 3]
b = [1, 2, 3]
c = a

# is：检查是否是同一个对象
print(a is c)      # True（c 和 a 指向同一对象）
print(a is b)      # False（a 和 b 是不同的对象，虽然内容相同）

# is not：检查是否不是同一个对象
print(a is not b)  # True
```

### is vs ==

```python
a = [1, 2, 3]
b = [1, 2, 3]

print(a == b)  # True（值相等）
print(a is b)  # False（不是同一对象）

# 对于不可变类型，Python 可能会优化
x = 256
y = 256
print(x is y)  # True（小整数会被缓存）

x = 1000
y = 1000
print(x is y)  # False（大整数不会被缓存）
```

> 💡 **最佳实践**：比较值用 `==`，检查是否为 None 用 `is`。

## 位运算符

用于对整数的二进制位进行操作（较少使用）。

```python
a = 12  # 二进制: 1100
b = 10  # 二进制: 1010

# 按位与
print(a & b)   # 8  （二进制: 1000）

# 按位或
print(a | b)   # 14 （二进制: 1110）

# 按位异或
print(a ^ b)   # 6  （二进制: 0110）

# 按位取反
print(~a)      # -13

# 左移
print(a << 2)  # 48 （12 * 2^2）

# 右移
print(a >> 2)  # 3  （12 / 2^2）
```

## 运算符优先级总览

从高到低：

1. `()` - 括号
2. `**` - 幂
3. `+x`, `-x`, `~x` - 正负号、取反
4. `*`, `/`, `//`, `%` - 乘除
5. `+`, `-` - 加减
6. `<<`, `>>` - 位移
7. `&` - 按位与
8. `^` - 按位异或
9. `|` - 按位或
10. `==`, `!=`, `>`, `<`, `>=`, `<=`, `is`, `in` - 比较
11. `not` - 逻辑非
12. `and` - 逻辑与
13. `or` - 逻辑或

> 💡 **提示**：不确定优先级时，使用括号明确表达意图。
