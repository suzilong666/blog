# 字符串

[官方文档](https://docs.python.org/zh-cn/3/library/stdtypes.html#text-sequence-type-str)

字符串是 Python 中最常用的数据类型之一，用于表示文本数据。Python 提供了丰富的字符串操作方法。本文将介绍字符串的创建、格式化、常用方法和切片操作。

## 字符串创建

### 基本语法

```python
# 单引号
single = 'Hello'

# 双引号
double = "World"

# 三引号（多行字符串）
multi = """这是
多行
字符串"""

print(single)   # Hello
print(double)   # World
print(multi)
# 这是
# 多行
# 字符串
```

### 转义字符

```python
# 常见转义字符
print("Hello\nWorld")    # 换行
print("Hello\tWorld")    # 制表符
print("He said \"Hi\"")  # 双引号
print('It\'s ok')        # 单引号
print("C:\\Users\\Name") # 反斜杠

# 原始字符串（不处理转义）
print(r"C:\Users\Name")  # C:\Users\Name
```

## 字符串格式化

### f-string（推荐，Python 3.6+）

```python
name = "张三"
age = 25

# 基本用法
greeting = f"你好，我叫{name}，今年{age}岁"
print(greeting)  # 你好，我叫张三，今年25岁

# 表达式计算
result = f"两年后我{age + 2}岁"
print(result)  # 两年后我27岁

# 格式化数字
pi = 3.1415926
print(f"π ≈ {pi:.2f}")    # π ≈ 3.14
print(f"π ≈ {pi:.4f}")    # π ≈ 3.1416

# 对齐和填充
name = "Python"
print(f"|{name:>10}|")    # |    Python|（右对齐）
print(f"|{name:<10}|")    # |Python    |（左对齐）
print(f"|{name:^10}|")    # |  Python  |（居中）
```

### format() 方法

```python
name = "李四"
age = 30

# 位置参数
message = "我叫{}，今年{}岁".format(name, age)
print(message)  # 我叫李四，今年30岁

# 索引参数
message = "{1}今年{0}岁".format(age, name)
print(message)  # 李四今年30岁

# 命名参数
message = "我叫{name}，今年{age}岁".format(name="王五", age=28)
print(message)  # 我叫王五，今年28岁
```

### % 格式化（旧式，不推荐）

```python
name = "赵六"
age = 35

message = "我叫%s，今年%d岁" % (name, age)
print(message)  # 我叫赵六，今年35岁
```

> 💡 **最佳实践**：优先使用 f-string，它更简洁、性能更好。

## 常用字符串方法

### 大小写转换

```python
text = "Hello, Python!"

print(text.upper())       # HELLO, PYTHON!
print(text.lower())       # hello, python!
print(text.title())       # Hello, Python!（每个单词首字母大写）
print(text.capitalize())  # Hello, python!（仅首字母大写）
print(text.swapcase())    # hELLO, pYTHON!（大小写互换）
```

### 查找和替换

```python
text = "Hello, World! Hello, Python!"

# 查找
print(text.find("Hello"))     # 0（首次出现的位置）
print(text.find("Java"))      # -1（未找到）
print(text.index("Hello"))    # 0
# print(text.index("Java"))   # ValueError!

# 计数
print(text.count("Hello"))    # 2

# 替换
new_text = text.replace("Hello", "Hi")
print(new_text)  # Hi, World! Hi, Python!

# 替换指定次数
new_text = text.replace("Hello", "Hi", 1)
print(new_text)  # Hi, World! Hello, Python!
```

### 判断方法

```python
text = "Hello123"

print(text.isalpha())    # False（包含数字）
print(text.isdigit())    # False（包含字母）
print(text.isalnum())    # True（字母或数字）
print(text.isupper())    # False（不是全大写）
print(text.islower())    # False（不是全小写）
print(text.isspace())    # False（不是空白）

# 前后缀检查
url = "https://example.com"
print(url.startswith("https"))  # True
print(url.endswith(".com"))     # True
```

### 去除空白

```python
text = "  Hello, World!  "

print(text.strip())    # "Hello, World!"（去除两端空白）
print(text.lstrip())   # "Hello, World!  "（去除左侧）
print(text.rstrip())   # "  Hello, World!"（去除右侧）

# 去除指定字符
text = "---Hello---"
print(text.strip("-"))  # "Hello"
```

### 分割和连接

```python
# 分割
text = "apple,banana,orange"
fruits = text.split(",")
print(fruits)  # ['apple', 'banana', 'orange']

# 按空格分割（默认）
text = "Hello World Python"
words = text.split()
print(words)  # ['Hello', 'World', 'Python']

# 限制分割次数
text = "a,b,c,d"
print(text.split(",", 2))  # ['a', 'b', 'c,d']

# 连接
fruits = ["apple", "banana", "orange"]
text = ", ".join(fruits)
print(text)  # apple, banana, orange
```

### 其他常用方法

```python
# 检查是否包含
text = "Hello, World!"
print("World" in text)     # True
print("Java" not in text)  # True

# 长度
print(len("Hello"))  # 5

# 重复
print("Ha" * 3)  # HaHaHa

# 填充
print("42".zfill(5))  # 00042

# 中心对齐
print("Python".center(20, "-"))  # -------Python-------
```

## 字符串切片

切片用于提取字符串的子串。

### 基本语法

```python
text = "Python Programming"

# text[start:end]：从 start 到 end-1
print(text[0:6])     # Python
print(text[7:18])    # Programming

# 省略 start 或 end
print(text[:6])      # Python（从头开始）
print(text[7:])      # Programming（到末尾）

# 负数索引
print(text[-11:])    # Programming（最后11个字符）
print(text[:-12])    # Py（除最后12个字符）
```

### 带步长的切片

```python
text = "0123456789"

# text[start:end:step]
print(text[::2])     # 02468（每隔一个字符）
print(text[::-1])    # 9876543210（反转字符串）
print(text[1:8:2])   # 1357
```

### 实际应用

```python
# 提取文件扩展名
filename = "document.pdf"
extension = filename.split(".")[-1]
print(extension)  # pdf

# 隐藏手机号中间位
phone = "13812345678"
hidden = phone[:3] + "****" + phone[7:]
print(hidden)  # 138****5678

# 反转字符串
text = "Hello"
reversed_text = text[::-1]
print(reversed_text)  # olleH
```

## 字符串不可变性

Python 字符串是不可变的，不能直接修改。

```python
text = "Hello"
# text[0] = "h"  # TypeError: 不支持赋值

# 需要创建新字符串
new_text = "h" + text[1:]
print(new_text)  # hello
```

## 常见错误

### 1. 索引越界

```python
text = "Hello"
# print(text[10])  # IndexError: 索引超出范围

# 安全做法
if len(text) > 10:
    print(text[10])
```

### 2. 类型拼接错误

```python
age = 25
# message = "我" + age + "岁"  # TypeError

# 正确做法
message = f"我{age}岁"
message = "我" + str(age) + "岁"
```

### 3. 忘记字符串不可变

```python
text = "Hello"
# text.replace("H", "h")  # 返回新字符串，原字符串不变

# 正确做法
text = text.replace("H", "h")
print(text)  # hello
```

## 实际应用示例

### 1. 统计词频

```python
def word_count(text):
    words = text.lower().split()
    count = {}
    for word in words:
        count[word] = count.get(word, 0) + 1
    return count

text = "hello world hello python world"
print(word_count(text))
# {'hello': 2, 'world': 2, 'python': 1}
```

### 2. 验证邮箱格式

```python
def is_valid_email(email):
    if "@" not in email:
        return False
    parts = email.split("@")
    if len(parts) != 2:
        return False
    username, domain = parts
    if not username or not domain:
        return False
    if "." not in domain:
        return False
    return True

print(is_valid_email("user@example.com"))  # True
print(is_valid_email("invalid"))           # False
```

### 3. 驼峰命名转换

```python
def to_snake_case(camel_case):
    result = []
    for i, char in enumerate(camel_case):
        if char.isupper() and i > 0:
            result.append("_")
        result.append(char.lower())
    return "".join(result)

print(to_snake_case("camelCase"))      # camel_case
print(to_snake_case("myVariableName")) # my_variable_name
```

## 最佳实践

1. **优先使用 f-string**：更简洁、性能更好
2. **使用 in 运算符**：检查子串存在性
3. **利用内置方法**：避免手动实现常见操作
4. **注意不可变性**：字符串操作返回新字符串
5. **使用 join() 拼接**：比 `+` 更高效（大量拼接时）
