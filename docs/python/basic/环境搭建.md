# 环境搭建

[官方文档](https://www.python.org/downloads/)

Python 是一种广泛使用的高级编程语言，以其简洁易读的语法和强大的生态系统而闻名。在开始学习 Python 之前，我们需要先搭建好开发环境。本文将介绍如何安装 Python、选择合适的开发工具，并运行你的第一个 Python 程序。

## 安装 Python

### Windows 系统

1. **下载安装包**
   - 访问 [Python 官网](https://www.python.org/downloads/)
   - 下载最新版本的 Python 3.x 安装包（推荐 Python 3.10+）

2. **运行安装程序**
   - 双击下载的安装包
   - **重要**：勾选 "Add Python to PATH" 选项
   - 点击 "Install Now" 开始安装

3. **验证安装**
   ```bash
   python --version
   # 或
   python3 --version
   ```
   如果显示版本号（如 `Python 3.11.4`），说明安装成功。

### macOS 系统

macOS 通常预装了 Python 2.x，但我们需要安装 Python 3.x。

1. **使用 Homebrew 安装**（推荐）
   ```bash
   # 先安装 Homebrew（如果未安装）
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

   # 安装 Python 3
   brew install python3
   ```

2. **验证安装**
   ```bash
   python3 --version
   ```

### Linux 系统

大多数 Linux 发行版已预装 Python 3。

1. **Ubuntu/Debian**
   ```bash
   sudo apt update
   sudo apt install python3 python3-pip
   ```

2. **CentOS/RHEL**
   ```bash
   sudo yum install python3 python3-pip
   ```

3. **验证安装**
   ```bash
   python3 --version
   ```

## 选择 Python 版本

目前推荐使用 **Python 3.10 或更高版本**，原因如下：

- **Python 2 已停止维护**：2020 年 1 月 1 日后不再接收安全更新
- **新特性支持**：Python 3.10+ 引入了模式匹配、更好的类型提示等新特性
- **库兼容性**：主流第三方库都已支持 Python 3

> 💡 **建议**：除非有特殊的遗留项目需求，否则始终使用最新的 Python 3 稳定版本。

## 第一个 Python 程序

### 交互式解释器

Python 自带交互式解释器，可以直接输入代码并立即看到结果。

```python
# 在终端输入 python3 进入交互模式
>>> print("Hello, World!")
Hello, World!
>>> 1 + 2
3
>>> exit()  # 退出交互模式
```

### 编写脚本文件

1. **创建文件**
   创建一个名为 `hello.py` 的文件：

   ```python
   # hello.py
   print("Hello, World!")

   # 变量和简单计算
   name = "Python"
   version = 3.11
   print(f"Welcome to {name} {version}!")
   ```

2. **运行程序**
   ```bash
   python3 hello.py
   ```

3. **输出结果**
   ```
   Hello, World!
   Welcome to Python 3.11!
   ```

## 包管理工具 pip

pip 是 Python 的官方包管理工具，用于安装和管理第三方库。

### 基本用法

```bash
# 查看 pip 版本
pip --version

# 安装第三方库
pip install requests

# 列出已安装的包
pip list

# 卸载包
pip uninstall requests
```

> 💡 **提示**：关于 pip 的详细使用方法，请参考 [pip 使用指南](/python/tools/pip)。

## 常见问题

### 1. "python" 命令找不到

**原因**：Python 未添加到系统 PATH 环境变量中。

**解决方案**：
- Windows：重新运行安装程序，确保勾选 "Add Python to PATH"
- macOS/Linux：使用 `python3` 命令代替 `python`

### 2. 多个 Python 版本共存

如果你的系统中同时存在 Python 2 和 Python 3：

```bash
# 明确指定使用 Python 3
python3 --version
pip3 install package_name
```

### 3. 权限问题

在某些系统上安装全局包可能需要管理员权限：

```bash
# 不推荐：使用 sudo
sudo pip3 install package_name

# 推荐：使用虚拟环境或 --user 标志
pip3 install --user package_name
```

> 💡 **最佳实践**：建议使用虚拟环境来隔离不同项目的依赖，避免权限问题和版本冲突。
