# 条件语句

[官方文档](https://docs.python.org/zh-cn/3/tutorial/controlflow.html#if-statements)

条件语句允许程序根据不同的条件执行不同的代码分支。Python 使用 `if`、`elif` 和 `else` 关键字来实现条件判断。本文将详细介绍 Python 的条件语句语法和用法。

## if 语句基础

### 基本语法

```python
age = 20

if age >= 18:
    print("成年人")
```

**注意**：
- 条件后面必须有冒号 `:`
- 代码块必须缩进（推荐 4 个空格）

### if-else 语句

```python
age = 16

if age >= 18:
    print("成年人")
else:
    print("未成年人")
# 输出: 未成年人
```

### if-elif-else 语句

```python
score = 85

if score >= 90:
    grade = "优秀"
elif score >= 80:
    grade = "良好"
elif score >= 70:
    grade = "中等"
elif score >= 60:
    grade = "及格"
else:
    grade = "不及格"

print(f"成绩等级：{grade}")  # 输出: 成绩等级：良好
```

> 💡 **提示**：`elif` 可以有多个，`else` 最多一个且可选。

## 真值测试

Python 中以下值被视为 `False`：

```python
# 布尔假值
print(bool(False))      # False
print(bool(None))       # False
print(bool(0))          # False
print(bool(0.0))        # False
print(bool(""))         # False（空字符串）
print(bool([]))         # False（空列表）
print(bool(()))         # False（空元组）
print(bool({}))         # False（空字典）
print(bool(set()))      # False（空集合）

# 其他所有值都为 True
print(bool(1))          # True
print(bool("hello"))    # True
print(bool([1, 2]))     # True
```

### 简化条件判断

```python
# 不推荐的写法
name = "Python"
if name != "":
    print(f"Hello, {name}")

# Pythonic 写法
if name:
    print(f"Hello, {name}")

# 检查列表是否为空
items = [1, 2, 3]
if items:  # 列表非空时为 True
    print(f"列表有 {len(items)} 个元素")
```

## 三元表达式

Python 的三元表达式（条件表达式）可以在一行中完成简单的条件判断。

### 基本语法

```python
# 传统写法
age = 20
if age >= 18:
    status = "成年"
else:
    status = "未成年"

# 三元表达式写法
status = "成年" if age >= 18 else "未成年"
print(status)  # 输出: 成年
```

### 实际应用

```python
# 获取最大值
a, b = 10, 20
max_value = a if a > b else b
print(max_value)  # 20

# 设置默认值
username = ""
display_name = username if username else "游客"
print(display_name)  # 游客

# 条件赋值
score = 75
result = "及格" if score >= 60 else "不及格"
print(result)  # 及格
```

## 嵌套条件

条件语句可以嵌套使用，但应避免过深的嵌套。

```python
age = 25
has_ticket = True

if age >= 18:
    if has_ticket:
        print("可以入场")
    else:
        print("请先购票")
else:
    print("未成年人禁止入内")
```

### 使用 and/or 简化嵌套

```python
# 上述嵌套可以简化为
if age >= 18 and has_ticket:
    print("可以入场")
elif age >= 18:
    print("请先购票")
else:
    print("未成年人禁止入内")
```

> 💡 **最佳实践**：嵌套不超过 3 层，过深时应考虑重构代码。

## match-case 语句（Python 3.10+）

Python 3.10 引入了模式匹配功能，类似于其他语言的 switch-case。

```python
def http_status(status_code):
    match status_code:
        case 200:
            return "OK"
        case 404:
            return "Not Found"
        case 500:
            return "Internal Server Error"
        case _:
            return "Unknown Status"

print(http_status(404))  # Not Found
```

### 匹配多种情况

```python
def get_day_type(day):
    match day:
        case "Saturday" | "Sunday":
            return "周末"
        case _:
            return "工作日"

print(get_day_type("Saturday"))  # 周末
```

## 常见错误

### 1. 忘记冒号

```python
# if age >= 18  # SyntaxError: 缺少冒号
#     print("成年")

# 正确写法
if age >= 18:
    print("成年")
```

### 2. 缩进错误

```python
# if True:
# print("Hello")  # IndentationError: 缺少缩进

# 正确写法
if True:
    print("Hello")
```

### 3. 混淆 = 和 ==

```python
x = 10

# if x = 5:  # SyntaxError: 不能在条件中使用赋值
#     pass

if x == 5:  # 正确：使用 == 进行比较
    print("x 等于 5")
```

### 4. 错误的逻辑

```python
# 错误：这个条件永远为 True
age = 25
# if age == 20 or 30:  # 30 被视为 True
#     print("匹配")

# 正确写法
if age == 20 or age == 30:
    print("匹配")

# 更简洁的写法
if age in (20, 30):
    print("匹配")
```

## 实际应用示例

### 1. 用户登录验证

```python
username = "admin"
password = "123456"

input_user = "admin"
input_pass = "123456"

if input_user == username and input_pass == password:
    print("登录成功")
elif input_user != username:
    print("用户名错误")
else:
    print("密码错误")
```

### 2. 成绩评级系统

```python
def get_grade(score):
    if not 0 <= score <= 100:
        return "无效分数"

    if score >= 90:
        return "A"
    elif score >= 80:
        return "B"
    elif score >= 70:
        return "C"
    elif score >= 60:
        return "D"
    else:
        return "F"

print(get_grade(85))  # B
```

### 3. 闰年判断

```python
def is_leap_year(year):
    if year % 4 != 0:
        return False
    elif year % 100 != 0:
        return True
    elif year % 400 != 0:
        return False
    else:
        return True

print(is_leap_year(2024))  # True
print(is_leap_year(1900))  # False
print(is_leap_year(2000))  # True
```

## 最佳实践

1. **保持条件简洁**：复杂条件提取为变量或函数
   ```python
   # 不好
   if age >= 18 and age <= 65 and has_job and income > 5000:
       pass

   # 好
   is_eligible = (18 <= age <= 65) and has_job and income > 5000
   if is_eligible:
       pass
   ```

2. **避免深层嵌套**：使用早期返回
   ```python
   # 不好
   def process(user):
       if user:
           if user.is_active:
               if user.has_permission:
                   return "OK"
       return "Denied"

   # 好
   def process(user):
       if not user:
           return "Denied"
       if not user.is_active:
           return "Denied"
       if not user.has_permission:
           return "Denied"
       return "OK"
   ```

3. **使用 in 运算符**：检查多个值
   ```python
   # 不好
   if day == "Monday" or day == "Tuesday" or day == "Wednesday":
       pass

   # 好
   if day in ("Monday", "Tuesday", "Wednesday"):
       pass
   ```
