# 装饰器

[官方文档](https://docs.python.org/zh-cn/3/glossary.html#term-decorator)

装饰器是 Python 中一种强大的设计模式，允许在不修改原函数代码的情况下，为函数添加额外功能。装饰器本质上是一个接受函数作为参数并返回新函数的高阶函数。本文将深入讲解装饰器的原理、用法和实际应用场景。

## 装饰器概念和原理

### 什么是装饰器？

装饰器是一种"包装"函数的机制，可以在不改变原函数代码的前提下：
- 在函数执行前添加预处理逻辑
- 在函数执行后添加后处理逻辑
- 修改函数的行为或返回值

### 基础原理

要理解装饰器，需要先理解两个核心概念：

1. **函数是一等公民**：函数可以赋值给变量、作为参数传递、作为返回值
2. **闭包**：内部函数可以访问外部函数的变量

```python
# 函数可以作为参数传递
def say_hello():
    print("Hello!")

def execute(func):
    func()

execute(say_hello)  # 输出: Hello!

# 函数可以作为返回值
def create_greeter(greeting):
    def greeter():
        print(greeting)
    return greeter

greet = create_greeter("Hi!")
greet()  # 输出: Hi!
```

## 函数装饰器基础语法

### 最简单的装饰器

```python
def my_decorator(func):
    """一个简单的装饰器"""
    def wrapper():
        print("函数执行前")
        func()
        print("函数执行后")
    return wrapper

# 使用装饰器
@my_decorator
def say_hello():
    print("Hello!")

say_hello()
# 输出:
# 函数执行前
# Hello!
# 函数执行后
```

`@my_decorator` 等价于 `say_hello = my_decorator(say_hello)`

### 装饰带参数的函数

```python
def my_decorator(func):
    def wrapper(*args, **kwargs):
        print("函数执行前")
        result = func(*args, **kwargs)
        print("函数执行后")
        return result
    return wrapper

@my_decorator
def greet(name):
    print(f"Hello, {name}!")

greet("Python")
# 输出:
# 函数执行前
# Hello, Python!
# 函数执行后
```

使用 `*args` 和 `**kwargs` 确保装饰器可以适配任何函数签名。

## functools.wraps 的作用

使用装饰器后，原函数的元信息（如函数名、文档字符串）会丢失：

```python
def my_decorator(func):
    def wrapper(*args, **kwargs):
        return func(*args, **kwargs)
    return wrapper

@my_decorator
def say_hello():
    """打招呼函数"""
    print("Hello!")

print(say_hello.__name__)  # 输出: wrapper（不是 say_hello）
print(say_hello.__doc__)   # 输出: None
```

使用 `functools.wraps` 保留原函数的元信息：

```python
from functools import wraps

def my_decorator(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        return func(*args, **kwargs)
    return wrapper

@my_decorator
def say_hello():
    """打招呼函数"""
    print("Hello!")

print(say_hello.__name__)  # 输出: say_hello
print(say_hello.__doc__)   # 输出: 打招呼函数
```

> 💡 **最佳实践**：始终在装饰器中使用 `@wraps(func)`。

## 带参数的装饰器

有时需要为装饰器本身传递参数，这需要三层嵌套：

```python
from functools import wraps

def repeat(times):
    """重复执行指定次数的装饰器"""
    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            for _ in range(times):
                result = func(*args, **kwargs)
            return result
        return wrapper
    return decorator

@repeat(times=3)
def greet(name):
    print(f"Hello, {name}!")

greet("Python")
# 输出:
# Hello, Python!
# Hello, Python!
# Hello, Python!
```

## 实际应用示例

### 1. 日志记录

```python
from functools import wraps
import time

def log_execution(func):
    """记录函数执行的日志"""
    @wraps(func)
    def wrapper(*args, **kwargs):
        print(f"[LOG] 调用函数: {func.__name__}")
        print(f"[LOG] 参数: args={args}, kwargs={kwargs}")

        start_time = time.time()
        result = func(*args, **kwargs)
        end_time = time.time()

        print(f"[LOG] 函数 {func.__name__} 执行耗时: {end_time - start_time:.4f}秒")
        print(f"[LOG] 返回值: {result}")

        return result
    return wrapper

@log_execution
def add(a, b):
    return a + b

add(3, 5)
```

### 2. 性能计时

```python
from functools import wraps
import time

def timer(func):
    """计算函数执行时间"""
    @wraps(func)
    def wrapper(*args, **kwargs):
        start = time.perf_counter()
        result = func(*args, **kwargs)
        end = time.perf_counter()
        print(f"{func.__name__} 执行时间: {end - start:.6f}秒")
        return result
    return wrapper

@timer
def slow_function():
    time.sleep(1)

slow_function()  # 输出: slow_function 执行时间: 1.000xxx秒
```

### 3. 缓存结果

```python
from functools import wraps

def cache(func):
    """简单的缓存装饰器"""
    cache_dict = {}

    @wraps(func)
    def wrapper(*args):
        if args not in cache_dict:
            cache_dict[args] = func(*args)
        return cache_dict[args]

    return wrapper

@cache
def fibonacci(n):
    """计算斐波那契数列"""
    if n < 2:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

print(fibonacci(10))  # 快速计算，因为有缓存
```

> 💡 **提示**：Python 标准库提供了 `@functools.lru_cache` 实现更完善的缓存功能。

### 4. 权限检查

```python
from functools import wraps

def require_login(func):
    """检查用户是否登录"""
    @wraps(func)
    def wrapper(user, *args, **kwargs):
        if not user.get("is_logged_in"):
            raise PermissionError("请先登录")
        return func(user, *args, **kwargs)
    return wrapper

@require_login
def view_profile(user):
    return f"欢迎, {user['name']}"

# 测试
logged_in_user = {"name": "张三", "is_logged_in": True}
print(view_profile(logged_in_user))  # 欢迎, 张三

guest_user = {"name": "游客", "is_logged_in": False}
# view_profile(guest_user)  # 抛出 PermissionError
```

## 类装饰器简介

除了函数装饰器，还可以使用类作为装饰器：

```python
class CountCalls:
    """统计函数调用次数的类装饰器"""
    def __init__(self, func):
        self.func = func
        self.call_count = 0

    def __call__(self, *args, **kwargs):
        self.call_count += 1
        print(f"{self.func.__name__} 被调用了 {self.call_count} 次")
        return self.func(*args, **kwargs)

@CountCalls
def say_hello():
    print("Hello!")

say_hello()  # say_hello 被调用了 1 次
say_hello()  # say_hello 被调用了 2 次
```

类装饰器适合需要维护状态的场景。

## 嵌套装饰器

可以同时使用多个装饰器，执行顺序是从下到上：

```python
from functools import wraps

def decorator_a(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        print("装饰器 A - 前")
        result = func(*args, **kwargs)
        print("装饰器 A - 后")
        return result
    return wrapper

def decorator_b(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        print("装饰器 B - 前")
        result = func(*args, **kwargs)
        print("装饰器 B - 后")
        return result
    return wrapper

@decorator_a
@decorator_b
def say_hello():
    print("Hello!")

say_hello()
# 输出:
# 装饰器 A - 前
# 装饰器 B - 前
# Hello!
# 装饰器 B - 后
# 装饰器 A - 后
```

## 内置装饰器

Python 提供了一些常用的内置装饰器：

### @staticmethod - 静态方法

```python
class MathUtils:
    @staticmethod
    def add(a, b):
        return a + b

# 不需要实例化即可调用
print(MathUtils.add(3, 5))  # 8
```

### @classmethod - 类方法

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    @classmethod
    def from_birth_year(cls, name, birth_year):
        """从出生年份创建对象"""
        age = 2024 - birth_year
        return cls(name, age)

person = Person.from_birth_year("张三", 1990)
print(person.age)  # 34
```

### @property - 属性装饰器

```python
class Circle:
    def __init__(self, radius):
        self.radius = radius

    @property
    def area(self):
        """计算面积（像访问属性一样调用方法）"""
        return 3.14159 * self.radius ** 2

circle = Circle(5)
print(circle.area)  # 78.53975（不需要加括号）
```

## 常见问题

### 1. 装饰器何时执行？

装饰器在**函数定义时**执行，而不是在函数调用时：

```python
def my_decorator(func):
    print("装饰器正在装饰函数")  # 这行在定义时执行
    return func

@my_decorator
def say_hello():  # 立即输出: 装饰器正在装饰函数
    print("Hello!")
```

### 2. 如何调试被装饰的函数？

使用 `@wraps` 保留元信息，或使用专门的调试装饰器：

```python
from functools import wraps

def debug(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        import pdb; pdb.set_trace()  # 设置断点
        return func(*args, **kwargs)
    return wrapper
```

## 最佳实践

1. **始终使用 `@wraps`**：保留原函数的元信息
2. **保持装饰器单一职责**：每个装饰器只做一件事
3. **使用描述性名称**：让装饰器用途一目了然
4. **考虑性能影响**：避免在高频调用的函数中使用复杂的装饰器
5. **文档化装饰器**：说明装饰器的作用和使用注意事项
