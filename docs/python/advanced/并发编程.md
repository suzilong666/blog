# 并发编程

[官方文档](https://docs.python.org/zh-cn/3/library/threading.html)

并发编程允许程序同时执行多个任务，提高效率和响应性。Python 提供了多线程和多进程两种并发模型。本文将介绍 threading 和 multiprocessing 模块的用法。

## threading 模块

### 创建线程

```python
import threading
import time

def task(name):
    print(f"线程 {name} 开始")
    time.sleep(2)
    print(f"线程 {name} 结束")

# 创建线程
thread = threading.Thread(target=task, args=("Thread-1",))

# 启动线程
thread.start()

# 等待线程完成
thread.join()
print("主线程继续")
```

### 多线程示例

```python
import threading
import time

def worker(num):
    print(f"工作线程 {num} 开始")
    time.sleep(1)
    print(f"工作线程 {num} 结束")

threads = []
for i in range(5):
    t = threading.Thread(target=worker, args=(i,))
    threads.append(t)
    t.start()

# 等待所有线程完成
for t in threads:
    t.join()

print("所有线程完成")
```

## 线程同步

### Lock（互斥锁）

```python
import threading

counter = 0
lock = threading.Lock()

def increment():
    global counter
    for _ in range(100000):
        with lock:  # 获取锁
            counter += 1
        # 释放锁

threads = [threading.Thread(target=increment) for _ in range(10)]
for t in threads:
    t.start()
for t in threads:
    t.join()

print(f"计数器: {counter}")  # 1000000
```

### RLock（可重入锁）

```python
import threading

rlock = threading.RLock()

def outer():
    with rlock:
        print("外层函数")
        inner()

def inner():
    with rlock:  # 同一线程可以再次获取
        print("内层函数")

outer()
```

### Condition（条件变量）

```python
import threading

condition = threading.Condition()
items = []

def producer():
    for i in range(5):
        with condition:
            items.append(i)
            print(f"生产: {i}")
            condition.notify()  # 通知消费者
        threading.Event().wait(0.1)

def consumer():
    for _ in range(5):
        with condition:
            while not items:
                condition.wait()  # 等待生产
            item = items.pop(0)
            print(f"消费: {item}")

t1 = threading.Thread(target=producer)
t2 = threading.Thread(target=consumer)
t1.start()
t2.start()
t1.join()
t2.join()
```

### Semaphore（信号量）

```python
import threading
import time

semaphore = threading.Semaphore(3)  # 最多 3 个并发

def task(num):
    with semaphore:
        print(f"任务 {num} 开始")
        time.sleep(1)
        print(f"任务 {num} 结束")

threads = [threading.Thread(target=task, args=(i,)) for i in range(10)]
for t in threads:
    t.start()
for t in threads:
    t.join()
```

## multiprocessing 模块

### 创建进程

```python
from multiprocessing import Process
import os

def task(name):
    print(f"进程 {name}, PID: {os.getpid()}")

process = Process(target=task, args=("Process-1",))
process.start()
process.join()
```

### 进程池

```python
from multiprocessing import Pool

def square(x):
    return x ** 2

if __name__ == "__main__":
    with Pool(processes=4) as pool:
        results = pool.map(square, range(10))
        print(results)  # [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
```

## GIL 影响

Python 的全局解释器锁（GIL）限制了同一时刻只有一个线程执行 Python 字节码。

```python
# CPU 密集型：多进程更好
# I/O 密集型：多线程足够

import threading
import time

def cpu_bound():
    total = 0
    for i in range(10000000):
        total += i

start = time.time()
threads = [threading.Thread(target=cpu_bound) for _ in range(4)]
for t in threads:
    t.start()
for t in threads:
    t.join()
print(f"多线程耗时: {time.time() - start:.2f}s")
```

## 常见错误

### 1. 忘记 join

```python
t = threading.Thread(target=task)
t.start()
# t.join()  # 忘记等待，程序可能提前退出
```

### 2. 死锁

```python
lock1 = threading.Lock()
lock2 = threading.Lock()

# 线程 1
with lock1:
    with lock2:  # 等待 lock2
        pass

# 线程 2
with lock2:
    with lock1:  # 等待 lock1，死锁！
        pass
```

## 最佳实践

1. **I/O 密集用线程**：网络请求、文件操作
2. **CPU 密集用进程**：计算密集型任务
3. **使用线程池/进程池**：管理资源
4. **避免共享状态**：使用队列通信
5. **注意 GIL 限制**：理解性能瓶颈
