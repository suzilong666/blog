# 函数基础

[官方文档](https://docs.python.org/zh-cn/3/tutorial/controlflow.html#defining-functions)

函数是组织好的、可重复使用的代码块，用于执行单一或相关联的功能。函数能提高代码的模块化和复用性，是 Python 编程的核心概念之一。本文将介绍如何定义和调用函数、参数传递机制以及函数的最佳实践。

## 定义函数

### 基本语法

使用 `def` 关键字定义函数：

```python
def greet():
    """打印问候语"""
    print("Hello, Python!")

# 调用函数
greet()  # 输出: Hello, Python!
```

函数定义的组成部分：
- `def`：定义函数的关键字
- `greet`：函数名（遵循变量命名规则）
- `()`：参数列表（可以为空）
- `:`：冒号表示函数头结束
- 缩进的代码块：函数体

### 文档字符串（Docstring）

文档字符串用于说明函数的功能，使用三引号包裹：

```python
def add(a, b):
    """
    计算两个数的和

    参数:
        a: 第一个数
        b: 第二个数

    返回:
        两个数的和
    """
    return a + b

# 查看文档字符串
print(add.__doc__)
```

> 💡 **最佳实践**：始终为函数编写清晰的文档字符串，特别是公共 API。

## 返回值

函数可以使用 `return` 语句返回值。

### 单个返回值

```python
def square(x):
    """计算平方"""
    return x ** 2

result = square(5)
print(result)  # 输出: 25
```

### 多个返回值

Python 函数可以返回多个值（实际上是返回一个元组）：

```python
def get_name_and_age():
    """返回姓名和年龄"""
    name = "张三"
    age = 25
    return name, age

# 解包返回值
name, age = get_name_and_age()
print(f"{name}今年{age}岁")  # 输出: 张三今年25岁
```

### 无返回值

如果函数没有 `return` 语句，默认返回 `None`：

```python
def say_hello():
    print("Hello!")

result = say_hello()
print(result)  # 输出: None
```

## 参数传递

### 位置参数

最常见的参数类型，按位置顺序传递：

```python
def introduce(name, age):
    """自我介绍"""
    print(f"我叫{name}，今年{age}岁")

introduce("李四", 30)  # 输出: 我叫李四，今年30岁
```

### 默认参数

为参数设置默认值，调用时可以省略：

```python
def greet(name, greeting="你好"):
    """带默认问候语的打招呼"""
    print(f"{greeting}，{name}！")

greet("王五")              # 输出: 你好，王五！
greet("赵六", "早上好")    # 输出: 早上好，赵六！
```

> ⚠️ **注意**：默认参数的陷阱 - 不要使用可变对象作为默认值：

```python
# 错误示例
def append_to_list(item, lst=[]):
    lst.append(item)
    return lst

print(append_to_list(1))  # [1]
print(append_to_list(2))  # [1, 2] - 不是预期的 [2]！

# 正确做法
def append_to_list(item, lst=None):
    if lst is None:
        lst = []
    lst.append(item)
    return lst
```

### 关键字参数

调用时明确指定参数名，提高可读性：

```python
def create_user(name, age, city="北京"):
    """创建用户信息"""
    return {"name": name, "age": age, "city": city}

# 使用关键字参数
user = create_user(name="张三", age=25, city="上海")
print(user)  # {'name': '张三', 'age': 25, 'city': '上海'}
```

## 作用域简介

变量的作用域决定了它在哪些地方可以访问。

### 局部变量

在函数内部定义的变量，只能在函数内部访问：

```python
def my_func():
    local_var = "我是局部变量"
    print(local_var)

my_func()  # 输出: 我是局部变量
# print(local_var)  # NameError: name 'local_var' is not defined
```

### 全局变量

在函数外部定义的变量，可以在整个模块中访问：

```python
global_var = "我是全局变量"

def my_func():
    print(global_var)  # 可以访问全局变量

my_func()  # 输出: 我是全局变量
```

### global 关键字

在函数内部修改全局变量需要使用 `global` 关键字：

```python
count = 0

def increment():
    global count
    count += 1

increment()
print(count)  # 输出: 1
```

> 💡 **最佳实践**：尽量避免在函数内部修改全局变量，优先使用参数传递和返回值。

## 常见错误示例

### 1. 忘记调用函数

```python
def greet():
    print("Hello!")

greet      # 只是引用函数对象，不会执行
greet()    # 正确：调用函数
```

### 2. 参数数量不匹配

```python
def add(a, b):
    return a + b

# add(1)           # TypeError: 缺少参数
# add(1, 2, 3)     # TypeError: 参数过多
add(1, 2)          # 正确
```

### 3. 混淆参数顺序

```python
def divide(dividend, divisor):
    return dividend / divisor

# 错误：顺序颠倒
result = divide(2, 10)   # 0.2，而不是 5.0

# 正确：使用关键字参数避免混淆
result = divide(dividend=10, divisor=2)  # 5.0
```

## 函数 vs 方法

在 Python 中，函数和方法有所区别：

```python
# 函数：独立定义
def my_function():
    pass

# 方法：属于类的函数
class MyClass:
    def my_method(self):
        pass
```

## 最佳实践

1. **单一职责原则**：每个函数只做一件事
   ```python
   # 不好：一个函数做太多事
   def process_data(data):
       # 验证、清洗、转换、保存...

   # 好：拆分为多个小函数
   def validate_data(data):
       pass

   def clean_data(data):
       pass

   def transform_data(data):
       pass
   ```

2. **函数命名清晰**：使用动词开头的描述性名称
   ```python
   def calc():          # 不好：不清楚计算什么
       pass

   def calculate_total():  # 好：明确表示计算总额
       pass
   ```

3. **保持函数简短**：理想情况下不超过 20-30 行

4. **使用类型提示**（Python 3.5+）：
   ```python
   def greet(name: str) -> str:
       return f"Hello, {name}!"
   ```

5. **避免副作用**：纯函数更容易测试和维护
