# 文件操作

[官方文档](https://docs.python.org/zh-cn/3/tutorial/inputoutput.html#reading-and-writing-files)

文件操作是编程中的重要技能，用于读取和写入数据。Python 提供了简洁的文件操作 API，支持文本文件、二进制文件以及 JSON、CSV 等格式。本文将详细介绍 Python 的文件操作方法。

## open() 函数

`open()` 是文件操作的核心函数。

```python
# 基本语法
file = open(filename, mode, encoding)

# 参数说明：
# filename: 文件路径
# mode: 打开模式
# encoding: 编码（文本文件推荐 utf-8）
```

### 打开模式

| 模式 | 说明 |
| :--- | :--- |
| `'r'` | 只读（默认） |
| `'w'` | 写入（覆盖） |
| `'a'` | 追加 |
| `'x'` | 创建（存在则失败） |
| `'b'` | 二进制模式 |
| `'t'` | 文本模式（默认） |
| `'+'` | 读写模式 |

```python
# 文本文件
f = open("example.txt", "r", encoding="utf-8")

# 二进制文件
f = open("image.png", "rb")

# 写入文件
f = open("output.txt", "w", encoding="utf-8")
```

## with 语句

使用 `with` 语句自动管理文件关闭，推荐使用。

```python
# 推荐做法
with open("example.txt", "r", encoding="utf-8") as f:
    content = f.read()
# 文件自动关闭

# 等价于
f = open("example.txt", "r", encoding="utf-8")
try:
    content = f.read()
finally:
    f.close()
```

## 读取文件

### 读取全部内容

```python
with open("example.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)
```

### 按行读取

```python
with open("example.txt", "r", encoding="utf-8") as f:
    # 读取一行
    line = f.readline()
    print(line)

    # 读取所有行（返回列表）
    lines = f.readlines()
    for line in lines:
        print(line.strip())  # strip() 去除换行符
```

### 逐行迭代（推荐）

```python
with open("example.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())
```

> 💡 **最佳实践**：大文件使用逐行迭代，避免一次性加载到内存。

## 写入文件

### 写入文本

```python
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("Hello, World!\n")
    f.write("第二行\n")
```

### 写入多行

```python
lines = ["第一行\n", "第二行\n", "第三行\n"]

with open("output.txt", "w", encoding="utf-8") as f:
    f.writelines(lines)
```

### 追加内容

```python
with open("output.txt", "a", encoding="utf-8") as f:
    f.write("追加的内容\n")
```

## 文件指针

```python
with open("example.txt", "r", encoding="utf-8") as f:
    # 当前位置
    print(f.tell())  # 0

    # 读取 10 个字符
    content = f.read(10)
    print(f.tell())  # 10

    # 移动指针
    f.seek(0)  # 回到开头
    print(f.tell())  # 0
```

## JSON 处理

JSON 是常用的数据交换格式。

```python
import json

# 字典转 JSON
data = {"name": "张三", "age": 25, "city": "北京"}
json_str = json.dumps(data, ensure_ascii=False, indent=2)
print(json_str)
# {
#   "name": "张三",
#   "age": 25,
#   "city": "北京"
# }

# JSON 转字典
parsed = json.loads(json_str)
print(parsed["name"])  # 张三
```

### 读写 JSON 文件

```python
import json

# 写入 JSON 文件
data = {"users": [{"name": "张三", "age": 25}]}

with open("data.json", "w", encoding="utf-8") as f:
    json.dump(data, f, ensure_ascii=False, indent=2)

# 读取 JSON 文件
with open("data.json", "r", encoding="utf-8") as f:
    data = json.load(f)
    print(data["users"][0]["name"])  # 张三
```

## CSV 处理

CSV 常用于表格数据。

```python
import csv

# 写入 CSV
data = [
    ["姓名", "年龄", "城市"],
    ["张三", 25, "北京"],
    ["李四", 30, "上海"]
]

with open("data.csv", "w", encoding="utf-8-sig", newline="") as f:
    writer = csv.writer(f)
    writer.writerows(data)

# 读取 CSV
with open("data.csv", "r", encoding="utf-8") as f:
    reader = csv.reader(f)
    for row in reader:
        print(row)
# ['姓名', '年龄', '城市']
# ['张三', '25', '北京']
# ['李四', '30', '上海']
```

### DictReader/DictWriter

```python
import csv

# 写入
data = [
    {"name": "张三", "age": 25, "city": "北京"},
    {"name": "李四", "age": 30, "city": "上海"}
]

with open("data.csv", "w", encoding="utf-8-sig", newline="") as f:
    writer = csv.DictWriter(f, fieldnames=["name", "age", "city"])
    writer.writeheader()
    writer.writerows(data)

# 读取
with open("data.csv", "r", encoding="utf-8") as f:
    reader = csv.DictReader(f)
    for row in reader:
        print(row["name"], row["age"])
```

## 路径操作

使用 `pathlib` 模块处理文件路径（Python 3.4+）。

```python
from pathlib import Path

# 创建路径对象
path = Path("folder/file.txt")

# 获取信息
print(path.name)       # file.txt
print(path.suffix)     # .txt
print(path.stem)       # file
print(path.parent)     # folder

# 检查存在性
print(path.exists())   # True/False
print(path.is_file())  # True/False
print(path.is_dir())   # True/False

# 遍历目录
for file in Path(".").glob("*.txt"):
    print(file.name)

# 创建目录
Path("new_folder").mkdir(exist_ok=True)

# 拼接路径
full_path = Path("folder") / "subfolder" / "file.txt"
print(full_path)  # folder/subfolder/file.txt
```

## 常见错误

### 1. 文件不存在

```python
# FileNotFoundError

# 安全做法
from pathlib import Path

path = Path("nonexistent.txt")
if path.exists():
    with open(path, "r") as f:
        content = f.read()
else:
    print("文件不存在")
```

### 2. 编码问题

```python
# UnicodeDecodeError

# 指定正确的编码
with open("chinese.txt", "r", encoding="utf-8") as f:
    content = f.read()
```

### 3. 忘记关闭文件

```python
# 不使用 with 时需要手动关闭
f = open("file.txt", "r")
try:
    content = f.read()
finally:
    f.close()

# 推荐：始终使用 with
with open("file.txt", "r") as f:
    content = f.read()
```

## 实际应用示例

### 1. 日志记录

```python
from datetime import datetime

def log_message(message, logfile="app.log"):
    timestamp = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
    with open(logfile, "a", encoding="utf-8") as f:
        f.write(f"[{timestamp}] {message}\n")

log_message("系统启动")
log_message("用户登录")
```

### 2. 配置文件读取

```python
import json
from pathlib import Path

def load_config(config_file="config.json"):
    path = Path(config_file)
    if not path.exists():
        return {}

    with open(path, "r", encoding="utf-8") as f:
        return json.load(f)

config = load_config()
print(config.get("debug", False))
```

### 3. 批量重命名文件

```python
from pathlib import Path

def rename_files(directory, old_ext, new_ext):
    dir_path = Path(directory)
    for file in dir_path.glob(f"*{old_ext}"):
        new_name = file.with_suffix(new_ext)
        file.rename(new_name)
        print(f"重命名: {file.name} -> {new_name.name}")

# rename_files("./photos", ".jpg", ".jpeg")
```

## 最佳实践

1. **始终使用 with 语句**：确保文件正确关闭
2. **指定编码**：文本文件使用 `encoding="utf-8"`
3. **使用 pathlib**：比 `os.path` 更现代
4. **异常处理**：捕获可能的 IO 错误
5. **大文件逐行读取**：避免内存溢出

```python
from pathlib import Path

def safe_read_file(filepath):
    path = Path(filepath)
    try:
        if not path.exists():
            raise FileNotFoundError(f"文件不存在: {filepath}")

        with open(path, "r", encoding="utf-8") as f:
            return f.read()
    except IOError as e:
        print(f"读取失败: {e}")
        return None
```
