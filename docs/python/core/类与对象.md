# 类与对象

[官方文档](https://docs.python.org/zh-cn/3/tutorial/classes.html)

在掌握类和对象的基础后，我们需要深入了解属性访问控制、不同类型的方法以及 property 装饰器。本文将详细介绍这些高级特性。

## 实例属性 vs 类属性

### 深入理解

```python
class Dog:
    species = "Canine"  # 类属性

    def __init__(self, name):
        self.name = name  # 实例属性

# 访问顺序：先查找实例属性，再查找类属性
dog = Dog("旺财")
print(dog.name)      # 旺财（实例属性）
print(dog.species)   # Canine（类属性）

# 通过实例修改类属性会创建新的实例属性
dog.species = "Dog"
print(dog.species)    # Dog（实例属性）
print(Dog.species)    # Canine（类属性未变）
```

### 实际应用

```python
class Counter:
    count = 0  # 类属性，追踪创建了多少个实例

    def __init__(self, name):
        self.name = name
        Counter.count += 1

c1 = Counter("A")
c2 = Counter("B")
print(Counter.count)  # 2
```

## 方法类型详解

### 实例方法

操作实例数据，需要 `self`。

```python
class Person:
    def __init__(self, name):
        self.name = name

    def greet(self):
        return f"你好，我是{self.name}"

person = Person("张三")
print(person.greet())  # 你好，我是张三
```

### 类方法

操作类数据，需要 `cls`，常用于工厂方法。

```python
class Date:
    def __init__(self, year, month, day):
        self.year = year
        self.month = month
        self.day = day

    @classmethod
    def from_string(cls, date_string):
        """从字符串创建日期对象"""
        year, month, day = map(int, date_string.split("-"))
        return cls(year, month, day)

    @classmethod
    def today(cls):
        """创建今天的日期对象"""
        import datetime
        now = datetime.date.today()
        return cls(now.year, now.month, now.day)

    def __str__(self):
        return f"{self.year}-{self.month:02d}-{self.day:02d}"

# 使用类方法创建对象
date1 = Date.from_string("2024-01-15")
print(date1)  # 2024-01-15

date2 = Date.today()
print(date2)  # 当前日期
```

### 静态方法

工具函数，不需要访问实例或类。

```python
class MathHelper:
    @staticmethod
    def is_prime(n):
        if n < 2:
            return False
        for i in range(2, int(n ** 0.5) + 1):
            if n % i == 0:
                return False
        return True

    @staticmethod
    def factorial(n):
        if n < 0:
            raise ValueError("负数没有阶乘")
        if n == 0:
            return 1
        result = 1
        for i in range(1, n + 1):
            result *= i
        return result

print(MathHelper.is_prime(17))     # True
print(MathHelper.factorial(5))     # 120
```

## 属性访问控制

Python 没有真正的私有属性，但有以下约定：

### 公开属性

```python
class Person:
    def __init__(self, name):
        self.name = name  # 公开

person = Person("张三")
print(person.name)  # 可以直接访问
```

### 受保护属性（约定）

单下划线前缀表示"内部使用"。

```python
class Person:
    def __init__(self, name):
        self._name = name  # 受保护（约定）

person = Person("张三")
print(person._name)  # 仍可访问，但不应该
```

### 名称改写（伪私有）

双下划线前缀触发名称改写。

```python
class Person:
    def __init__(self, name):
        self.__name = name  # 名称改写

person = Person("张三")
# print(person.__name)  # AttributeError!

# 实际名称被改写为 _ClassName__attribute
print(person._Person__name)  # 张三（不推荐这样做）
```

> 💡 **提示**：名称改写主要用于避免子类命名冲突，不是真正的私有。

## property 装饰器

property 允许将方法伪装成属性，实现 getter/setter。

### 基本用法

```python
class Circle:
    def __init__(self, radius):
        self.radius = radius

    @property
    def area(self):
        """计算面积（只读属性）"""
        return 3.14159 * self.radius ** 2

    @property
    def circumference(self):
        """计算周长（只读属性）"""
        return 2 * 3.14159 * self.radius

circle = Circle(5)
print(circle.area)           # 78.53975（像访问属性一样）
print(circle.circumference)  # 31.4159
```

### Getter 和 Setter

```python
class Temperature:
    def __init__(self, celsius):
        self.celsius = celsius

    @property
    def celsius(self):
        return self._celsius

    @celsius.setter
    def celsius(self, value):
        if value < -273.15:
            raise ValueError("温度不能低于绝对零度")
        self._celsius = value

    @property
    def fahrenheit(self):
        return self._celsius * 9 / 5 + 32

    @fahrenheit.setter
    def fahrenheit(self, value):
        self.celsius = (value - 32) * 5 / 9

# 使用
temp = Temperature(25)
print(temp.celsius)      # 25
print(temp.fahrenheit)   # 77.0

temp.fahrenheit = 100
print(temp.celsius)      # 37.777...
```

### 删除器

```python
class Person:
    def __init__(self, name):
        self._name = name

    @property
    def name(self):
        return self._name

    @name.setter
    def name(self, value):
        if not value:
            raise ValueError("姓名不能为空")
        self._name = value

    @name.deleter
    def name(self):
        del self._name

person = Person("张三")
print(person.name)  # 张三
del person.name     # 删除属性
# print(person.name)  # AttributeError
```

## 完整示例

```python
class BankAccount:
    """银行账户"""

    def __init__(self, owner, balance=0):
        self.owner = owner
        self._balance = balance  # 受保护
        self._transaction_history = []

    @property
    def balance(self):
        """查询余额（只读）"""
        return self._balance

    def deposit(self, amount):
        """存款"""
        if amount <= 0:
            raise ValueError("金额必须为正数")
        self._balance += amount
        self._transaction_history.append(f"存入: {amount}")
        return self._balance

    def withdraw(self, amount):
        """取款"""
        if amount <= 0:
            raise ValueError("金额必须为正数")
        if amount > self._balance:
            raise ValueError("余额不足")
        self._balance -= amount
        self._transaction_history.append(f"取出: {amount}")
        return self._balance

    @property
    def history(self):
        """交易历史（只读）"""
        return self._transaction_history.copy()

    def __str__(self):
        return f"{self.owner}的账户，余额: {self._balance}"

# 使用
account = BankAccount("张三", 1000)
account.deposit(500)
account.withdraw(200)
print(account.balance)   # 1300（通过 property 访问）
print(account.history)   # ['存入: 500', '取出: 200']
# account.balance = 999  # AttributeError!（不能直接修改）
```

## 常见错误

### 1. 忘记调用 property

```python
class Circle:
    def __init__(self, radius):
        self.radius = radius

    def area(self):  # 这是方法，不是属性
        return 3.14159 * self.radius ** 2

circle = Circle(5)
# print(circle.area)   # <bound method Circle.area>
print(circle.area())  # 78.53975（需要加括号）
```

### 2. setter 中无限递归

```python
class Temperature:
    @property
    def celsius(self):
        return self.celsius  # 无限递归！

    @celsius.setter
    def celsius(self, value):
        self.celsius = value  # 无限递归！

# 正确做法：使用不同的内部变量名
class Temperature:
    def __init__(self):
        self._celsius = 0

    @property
    def celsius(self):
        return self._celsius

    @celsius.setter
    def celsius(self, value):
        self._celsius = value
```

### 3. 混淆类方法和静态方法

```python
# 需要访问类数据时用类方法
class MyClass:
    count = 0

    @classmethod
    def get_count(cls):
        return cls.count  # 可以访问类属性

    @staticmethod
    def utility():
        # 不能访问 cls 或 self
        return "utility"
```

## 最佳实践

1. **使用 property 验证数据**：在 setter 中添加验证逻辑
2. **只读属性用 property**：不提供 setter
3. **工厂方法用类方法**：提供多种创建对象的方式
4. **工具函数用静态方法**：不依赖实例或类状态
5. **谨慎使用名称改写**：通常单下划线就够了

```python
class Rectangle:
    def __init__(self, width, height):
        self.width = width
        self.height = height

    @property
    def width(self):
        return self._width

    @width.setter
    def width(self, value):
        if value <= 0:
            raise ValueError("宽度必须为正数")
        self._width = value

    @property
    def area(self):
        return self._width * self.height
```
