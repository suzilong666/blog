# 面向对象基础

[官方文档](https://docs.python.org/zh-cn/3/tutorial/classes.html)

面向对象编程（OOP）是一种编程范式，通过将数据和操作数据的方法封装在对象中来组织代码。Python 是一门支持面向对象的编程语言。本文将介绍类和对象的基本概念。

## 类和对象概念

### 什么是类？

类是创建对象的蓝图或模板，定义了对象的属性和行为。

```python
# 类比：类是"狗"的概念，对象是具体的"旺财"
class Dog:
    pass
```

### 什么是对象？

对象是类的实例，具有类定义的属性和行为。

```python
# 创建对象（实例化）
my_dog = Dog()
your_dog = Dog()
```

## class 定义

### 基本语法

```python
class ClassName:
    """类的文档字符串"""

    # 类属性
    species = "Canine"

    # 方法
    def bark(self):
        print("汪汪!")

# 使用
dog = ClassName()
dog.bark()  # 汪汪!
```

### 命名规范

- 类名使用 **大驼峰命名法**（`MyClass`）
- 方法和属性使用 **小写+下划线**（`my_method`）

## __init__ 构造方法

`__init__` 是初始化方法，在创建对象时自动调用。

```python
class Person:
    def __init__(self, name, age):
        self.name = name  # 实例属性
        self.age = age

# 创建对象时传入参数
person = Person("张三", 25)
print(person.name)  # 张三
print(person.age)   # 25
```

### self 参数

`self` 代表实例本身，必须作为方法的第一个参数。

```python
class Person:
    def __init__(self, name):
        self.name = name

    def greet(self):
        # self 访问实例属性
        print(f"你好，我叫{self.name}")

person = Person("李四")
person.greet()  # 你好，我叫李四
```

> 💡 **提示**：`self` 是约定俗成的名称，也可以使用其他名字，但不推荐。

## 实例属性 vs 类属性

```python
class Dog:
    species = "Canine"  # 类属性（所有实例共享）

    def __init__(self, name):
        self.name = name  # 实例属性（每个实例独立）

dog1 = Dog("旺财")
dog2 = Dog("小白")

print(dog1.name)     # 旺财
print(dog2.name)     # 小白
print(dog1.species)  # Canine
print(dog2.species)  # Canine

# 修改类属性影响所有实例
Dog.species = "Dog"
print(dog1.species)  # Dog
print(dog2.species)  # Dog

# 修改实例属性只影响当前实例
dog1.name = "大黄"
print(dog1.name)  # 大黄
print(dog2.name)  # 小白
```

## 方法类型

### 实例方法

操作实例数据，第一个参数是 `self`。

```python
class Person:
    def __init__(self, name):
        self.name = name

    def greet(self):
        print(f"你好，我是{self.name}")

person = Person("张三")
person.greet()  # 你好，我是张三
```

### 类方法

操作类数据，第一个参数是 `cls`，使用 `@classmethod` 装饰器。

```python
class Person:
    count = 0  # 类属性

    def __init__(self, name):
        self.name = name
        Person.count += 1

    @classmethod
    def get_count(cls):
        return cls.count

p1 = Person("张三")
p2 = Person("李四")
print(Person.get_count())  # 2
```

### 静态方法

不操作实例或类数据，使用 `@staticmethod` 装饰器。

```python
class MathUtils:
    @staticmethod
    def add(a, b):
        return a + b

    @staticmethod
    def is_even(number):
        return number % 2 == 0

print(MathUtils.add(3, 5))      # 8
print(MathUtils.is_even(4))     # True
```

## 完整示例

```python
class BankAccount:
    """银行账户类"""

    # 类属性
    bank_name = "Python Bank"
    interest_rate = 0.03

    def __init__(self, owner, balance=0):
        # 实例属性
        self.owner = owner
        self.balance = balance

    def deposit(self, amount):
        """存款"""
        if amount <= 0:
            raise ValueError("存款金额必须为正数")
        self.balance += amount
        print(f"存入 {amount}，余额: {self.balance}")

    def withdraw(self, amount):
        """取款"""
        if amount <= 0:
            raise ValueError("取款金额必须为正数")
        if amount > self.balance:
            raise ValueError("余额不足")
        self.balance -= amount
        print(f"取出 {amount}，余额: {self.balance}")

    def get_balance(self):
        """查询余额"""
        return self.balance

    def __str__(self):
        return f"{self.owner}的账户，余额: {self.balance}"

# 使用
account = BankAccount("张三", 1000)
account.deposit(500)    # 存入 500，余额: 1500
account.withdraw(200)   # 取出 200，余额: 1300
print(account.get_balance())  # 1300
print(account)  # 张三的账户，余额: 1300
```

## 常见错误

### 1. 忘记 self

```python
class Person:
    def __init__(name, age):  # 错误！缺少 self
        name = name
        age = age

# 正确
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
```

### 2. 混淆实例属性和类属性

```python
class Dog:
    tricks = []  # 类属性（所有实例共享）

    def __init__(self, name):
        self.name = name

    def add_trick(self, trick):
        self.tricks.append(trick)

dog1 = Dog("旺财")
dog2 = Dog("小白")

dog1.add_trick("坐下")
print(dog2.tricks)  # ['坐下']（意外共享！）

# 正确做法：使用实例属性
class Dog:
    def __init__(self, name):
        self.name = name
        self.tricks = []  # 实例属性
```

### 3. 直接访问私有属性（Python 没有真正的私有）

```python
class Person:
    def __init__(self, name):
        self._name = name  # 约定：单下划线表示"受保护"

# Python 没有真正的私有属性，只是约定
```

## 实际应用示例

### 1. 矩形类

```python
class Rectangle:
    def __init__(self, width, height):
        self.width = width
        self.height = height

    def area(self):
        return self.width * self.height

    def perimeter(self):
        return 2 * (self.width + self.height)

    def is_square(self):
        return self.width == self.height

rect = Rectangle(5, 10)
print(f"面积: {rect.area()}")         # 面积: 50
print(f"周长: {rect.perimeter()}")    # 周长: 30
print(f"正方形: {rect.is_square()}")  # 正方形: False
```

### 2. 学生管理系统

```python
class Student:
    def __init__(self, student_id, name):
        self.student_id = student_id
        self.name = name
        self.grades = {}

    def add_grade(self, subject, grade):
        self.grades[subject] = grade

    def get_average(self):
        if not self.grades:
            return 0
        return sum(self.grades.values()) / len(self.grades)

    def __str__(self):
        avg = self.get_average()
        return f"{self.name}({self.student_id}) - 平均分: {avg:.2f}"

# 使用
student = Student("001", "张三")
student.add_grade("数学", 90)
student.add_grade("英语", 85)
print(student)  # 张三(001) - 平均分: 87.50
```

## 最佳实践

1. **明确的职责**：每个类只做一件事
2. **封装数据**：通过方法访问和修改属性
3. **合理的命名**：类名用名词，方法名用动词
4. **文档字符串**：为类和方法添加说明
5. **避免过度设计**：简单场景不需要复杂的类层次

```python
class Circle:
    """圆形类"""

    def __init__(self, radius):
        if radius < 0:
            raise ValueError("半径不能为负数")
        self.radius = radius

    @property
    def area(self):
        """计算面积"""
        return 3.14159 * self.radius ** 2

    @property
    def circumference(self):
        """计算周长"""
        return 2 * 3.14159 * self.radius
```
