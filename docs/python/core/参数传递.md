# 参数传递

[官方文档](https://docs.python.org/zh-cn/3/tutorial/controlflow.html#more-on-defining-functions)

Python 函数的参数传递机制非常灵活，支持位置参数、关键字参数、默认参数以及可变参数。理解这些参数类型及其特性对于编写优雅的 Python 代码至关重要。本文将深入讲解 Python 的参数传递机制。

## 位置参数

位置参数是最常见的参数类型，调用时按顺序传递。

```python
def greet(name, age):
    """打招呼"""
    print(f"我叫{name}，今年{age}岁")

# 按位置传递
greet("张三", 25)  # 我叫张三，今年25岁
greet(25, "张三")  # 我叫25，今年张三岁（顺序错误！）
```

> ⚠️ **注意**：位置参数的顺序必须与定义一致。

## 关键字参数

调用时明确指定参数名，提高可读性。

```python
def create_user(name, age, city):
    """创建用户"""
    return {"name": name, "age": age, "city": city}

# 使用关键字参数
user = create_user(name="李四", age=30, city="北京")
print(user)  # {'name': '李四', 'age': 30, 'city': '北京'}

# 可以改变顺序
user = create_user(city="上海", name="王五", age=28)
print(user)  # {'name': '王五', 'age': 28, 'city': '上海'}
```

### 混合使用

```python
def introduce(name, age, city="未知"):
    print(f"{name}, {age}岁, 来自{city}")

# 位置参数 + 关键字参数
introduce("赵六", 35, city="广州")  # 正确

# 位置参数必须在关键字参数之前
# introduce(name="孙七", 40, "深圳")  # SyntaxError!
introduce("孙七", 40, city="深圳")  # 正确
```

## 默认参数

为参数设置默认值，调用时可以省略。

```python
def power(base, exponent=2):
    """计算幂，默认平方"""
    return base ** exponent

print(power(5))      # 25（5的平方）
print(power(5, 3))   # 125（5的立方）
```

### 默认参数的陷阱

**重要**：不要使用可变对象作为默认参数！

```python
# ❌ 错误示例
def append_to_list(item, lst=[]):
    lst.append(item)
    return lst

print(append_to_list(1))  # [1]
print(append_to_list(2))  # [1, 2]（不是预期的 [2]！）
print(append_to_list(3))  # [1, 2, 3]

# ✅ 正确做法
def append_to_list(item, lst=None):
    if lst is None:
        lst = []
    lst.append(item)
    return lst

print(append_to_list(1))  # [1]
print(append_to_list(2))  # [2]
print(append_to_list(3))  # [3]
```

**原因**：默认参数在函数定义时只创建一次，所有调用共享同一个对象。

## *args - 可变位置参数

`*args` 允许函数接受任意数量的位置参数。

```python
def sum_all(*args):
    """计算所有参数的和"""
    total = 0
    for num in args:
        total += num
    return total

print(sum_all(1, 2, 3))       # 6
print(sum_all(1, 2, 3, 4, 5)) # 15
print(sum_all())              # 0

# args 是一个元组
def show_args(*args):
    print(f"类型: {type(args)}")
    print(f"内容: {args}")

show_args(1, 2, 3)
# 类型: <class 'tuple'>
# 内容: (1, 2, 3)
```

### 实际应用

```python
def concatenate(*strings):
    """拼接多个字符串"""
    return ", ".join(strings)

print(concatenate("apple", "banana", "orange"))
# apple, banana, orange

def max_value(*numbers):
    """找出最大值"""
    if not numbers:
        return None
    max_num = numbers[0]
    for num in numbers[1:]:
        if num > max_num:
            max_num = num
    return max_num

print(max_value(3, 1, 4, 1, 5, 9))  # 9
```

## **kwargs - 可变关键字参数

`**kwargs` 允许函数接受任意数量的关键字参数。

```python
def print_info(**kwargs):
    """打印所有关键字参数"""
    print(f"类型: {type(kwargs)}")
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="张三", age=25, city="北京")
# 类型: <class 'dict'>
# name: 张三
# age: 25
# city: 北京
```

### 实际应用

```python
def create_profile(**kwargs):
    """创建用户资料"""
    profile = {"name": "未知", "age": 0}
    profile.update(kwargs)
    return profile

user = create_profile(name="李四", age=30, email="lisi@example.com")
print(user)
# {'name': '李四', 'age': 30, 'email': 'lisi@example.com'}
```

## 组合使用

可以同时使用多种参数类型，但顺序必须正确：

```python
def func(positional, default="默认", *args, **kwargs):
    print(f"位置参数: {positional}")
    print(f"默认参数: {default}")
    print(f"可变位置参数: {args}")
    print(f"可变关键字参数: {kwargs}")

func("必传", "自定义", 1, 2, 3, key1="value1", key2="value2")
# 位置参数: 必传
# 默认参数: 自定义
# 可变位置参数: (1, 2, 3)
# 可变关键字参数: {'key1': 'value1', 'key2': 'value2'}
```

**参数顺序规则**：
1. 位置参数
2. 默认参数
3. `*args`
4. `**kwargs`

## 解包参数

调用函数时可以解包列表或字典。

```python
def greet(name, age, city):
    print(f"{name}, {age}岁, 来自{city}")

# 解包列表
args = ["张三", 25, "北京"]
greet(*args)  # 张三, 25岁, 来自北京

# 解包字典
kwargs = {"name": "李四", "age": 30, "city": "上海"}
greet(**kwargs)  # 李四, 30岁, 来自上海
```

### 实际应用

```python
def add(a, b, c):
    return a + b + c

# 从列表解包
numbers = [1, 2, 3]
print(add(*numbers))  # 6

# 从字典解包
params = {"a": 10, "b": 20, "c": 30}
print(add(**params))  # 60
```

## 强制关键字参数

`*` 之后的参数必须使用关键字传递。

```python
def create_user(name, *, age, city):
    """age 和 city 必须使用关键字参数"""
    return {"name": name, "age": age, "city": city}

# create_user("张三", 25, "北京")  # TypeError!
user = create_user("张三", age=25, city="北京")  # 正确
print(user)  # {'name': '张三', 'age': 25, 'city': '北京'}
```

## 强制位置参数

Python 3.8+ 支持 `/` 之前的参数必须使用位置传递。

```python
def divide(a, b, /):
    """a 和 b 必须使用位置参数"""
    return a / b

print(divide(10, 2))  # 5.0
# divide(a=10, b=2)  # TypeError!
```

## 常见错误

### 1. 参数数量不匹配

```python
def greet(name, age):
    print(f"{name}, {age}岁")

# greet("张三")          # TypeError: 缺少参数
# greet("张三", 25, "北京")  # TypeError: 参数过多
greet("张三", 25)  # 正确
```

### 2. 重复传参

```python
def greet(name, age):
    print(f"{name}, {age}岁")

# greet("张三", name="李四")  # TypeError: 重复赋值
```

### 3. 可变默认参数陷阱

```python
# ❌ 危险
def add_item(item, items=[]):
    items.append(item)
    return items

# ✅ 安全
def add_item(item, items=None):
    if items is None:
        items = []
    items.append(item)
    return items
```

## 实际应用示例

### 1. 灵活的日志函数

```python
def log(message, level="INFO", *args, **kwargs):
    """灵活的日志函数"""
    timestamp = kwargs.get("timestamp", True)
    prefix = f"[{level}]"

    if timestamp:
        import datetime
        now = datetime.datetime.now().strftime("%Y-%m-%d %H:%M:%S")
        prefix = f"{now} {prefix}"

    print(f"{prefix} {message}")
    if args:
        print(f"  额外参数: {args}")

log("系统启动")
log("错误发生", "ERROR", "details", timestamp=False)
```

### 2. URL 构建器

```python
def build_url(base, **params):
    """构建带查询参数的 URL"""
    if not params:
        return base

    query_string = "&".join(f"{k}={v}" for k, v in params.items())
    return f"{base}?{query_string}"

url = build_url("https://api.example.com/users", page=1, limit=10, sort="name")
print(url)
# https://api.example.com/users?page=1&limit=10&sort=name
```

### 3. 配置合并

```python
def merge_config(default_config, **user_config):
    """合并默认配置和用户配置"""
    config = default_config.copy()
    config.update(user_config)
    return config

default = {"host": "localhost", "port": 8080, "debug": False}
custom = merge_config(default, port=3000, debug=True)
print(custom)
# {'host': 'localhost', 'port': 3000, 'debug': True}
```

## 最佳实践

1. **优先使用关键字参数**：提高代码可读性
2. **避免过多参数**：超过 5 个考虑使用字典或配置对象
3. **谨慎使用可变默认参数**：始终用 `None` 代替
4. **合理使用 *args/**kwargs**：不要滥用，保持接口清晰
5. **添加类型提示**：帮助理解参数类型

```python
# 好的实践
def create_user(
    name: str,
    age: int,
    email: str | None = None,
    **kwargs
) -> dict:
    """创建用户"""
    user = {"name": name, "age": age}
    if email:
        user["email"] = email
    user.update(kwargs)
    return user
```
