# 继承与多态

[官方文档](https://docs.python.org/zh-cn/3/tutorial/classes.html#inheritance)

继承是面向对象编程的核心特性之一，允许我们基于现有类创建新类，实现代码复用和扩展。多态则让不同类的对象可以以统一的方式处理。本文将详细介绍 Python 的继承和多态机制。

## 单继承

### 基本语法

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        raise NotImplementedError("子类必须实现此方法")

# Dog 继承自 Animal
class Dog(Animal):
    def speak(self):
        return f"{self.name}说: 汪汪!"

class Cat(Animal):
    def speak(self):
        return f"{self.name}说: 喵喵!"

# 使用
dog = Dog("旺财")
print(dog.speak())  # 旺财说: 汪汪!

cat = Cat("小白")
print(cat.speak())  # 小白说: 喵喵!
```

### 扩展父类功能

```python
class Animal:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def info(self):
        return f"{self.name}, {self.age}岁"

class Dog(Animal):
    def __init__(self, name, age, breed):
        super().__init__(name, age)  # 调用父类构造方法
        self.breed = breed  # 新增属性

    def info(self):
        base_info = super().info()  # 调用父类方法
        return f"{base_info}, 品种: {self.breed}"

dog = Dog("旺财", 3, "金毛")
print(dog.info())  # 旺财, 3岁, 品种: 金毛
```

## super() 函数

`super()` 用于调用父类的方法。

```python
class Rectangle:
    def __init__(self, width, height):
        self.width = width
        self.height = height

    def area(self):
        return self.width * self.height

class Square(Rectangle):
    def __init__(self, side):
        super().__init__(side, side)  # 正方形是特殊的矩形

square = Square(5)
print(square.area())  # 25
```

### 多层继承

```python
class A:
    def method(self):
        return "A"

class B(A):
    def method(self):
        return "B -> " + super().method()

class C(B):
    def method(self):
        return "C -> " + super().method()

c = C()
print(c.method())  # C -> B -> A
```

## 多继承

Python 支持一个类继承多个父类。

```python
class Flyable:
    def fly(self):
        return f"{self.name}在飞"

class Swimmable:
    def swim(self):
        return f"{self.name}在游泳"

class Duck(Flyable, Swimmable):
    def __init__(self, name):
        self.name = name

duck = Duck("唐老鸭")
print(duck.fly())   # 唐老鸭在飞
print(duck.swim())  # 唐老鸭在游泳
```

### MRO 方法解析顺序

Python 使用 C3 线性化算法确定方法调用顺序。

```python
class A:
    def method(self):
        return "A"

class B(A):
    pass

class C(A):
    def method(self):
        return "C"

class D(B, C):
    pass

d = D()
print(d.method())  # C（按照 MRO 查找）

# 查看 MRO
print(D.__mro__)
# (<class 'D'>, <class 'B'>, <class 'C'>, <class 'A'>, <class 'object'>)
```

## 方法重写

子类可以重写父类的方法。

```python
class Animal:
    def speak(self):
        return "..."

class Dog(Animal):
    def speak(self):  # 重写
        return "汪汪!"

class Cat(Animal):
    def speak(self):  # 重写
        return "喵喵!"

animals = [Dog(), Cat()]
for animal in animals:
    print(animal.speak())
# 汪汪!
# 喵喵!
```

## 多态

多态允许不同类的对象对同一消息做出不同响应。

### 鸭子类型

"如果它走起来像鸭子，叫起来像鸭子，那它就是鸭子。"

```python
class Duck:
    def quack(self):
        return "嘎嘎!"

class Person:
    def quack(self):
        return "我在模仿鸭子叫"

def make_it_quack(thing):
    print(thing.quack())

make_it_quack(Duck())   # 嘎嘎!
make_it_quack(Person()) # 我在模仿鸭子叫
```

### 实际示例

```python
class Payment:
    def pay(self, amount):
        raise NotImplementedError

class Alipay(Payment):
    def pay(self, amount):
        return f"支付宝支付: {amount}元"

class WechatPay(Payment):
    def pay(self, amount):
        return f"微信支付: {amount}元"

class CreditCard(Payment):
    def pay(self, amount):
        return f"信用卡支付: {amount}元"

# 多态：统一接口，不同实现
def process_payment(payment_method: Payment, amount):
    print(payment_method.pay(amount))

process_payment(Alipay(), 100)      # 支付宝支付: 100元
process_payment(WechatPay(), 200)   # 微信支付: 200元
process_payment(CreditCard(), 300)  # 信用卡支付: 300元
```

## 抽象基类

使用 `abc` 模块定义抽象基类，强制子类实现特定方法。

```python
from abc import ABC, abstractmethod

class Shape(ABC):
    @abstractmethod
    def area(self):
        pass

    @abstractmethod
    def perimeter(self):
        pass

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius

    def area(self):
        return 3.14159 * self.radius ** 2

    def perimeter(self):
        return 2 * 3.14159 * self.radius

# circle = Shape()  # TypeError! 不能实例化抽象类
circle = Circle(5)
print(circle.area())      # 78.53975
print(circle.perimeter()) # 31.4159
```

## isinstance 和 issubclass

```python
class Animal:
    pass

class Dog(Animal):
    pass

dog = Dog()

# 检查实例类型
print(isinstance(dog, Dog))     # True
print(isinstance(dog, Animal))  # True（继承关系）
print(isinstance(dog, object))  # True

# 检查类继承关系
print(issubclass(Dog, Animal))  # True
print(issubclass(Animal, Dog))  # False
```

## 常见错误

### 1. 忘记调用 super().__init__()

```python
class Animal:
    def __init__(self, name):
        self.name = name

class Dog(Animal):
    def __init__(self, name, breed):
        # 忘记调用 super().__init__(name)
        self.breed = breed

# dog = Dog("旺财", "金毛")
# print(dog.name)  # AttributeError!
```

### 2. 钻石继承问题

```python
class A:
    def method(self):
        return "A"

class B(A):
    def method(self):
        return "B"

class C(A):
    def method(self):
        return "C"

class D(B, C):
    pass

# Python 的 MRO 解决了这个问题
d = D()
print(d.method())  # B（按照 MRO）
```

## 实际应用示例

### 1. 员工系统

```python
class Employee:
    def __init__(self, name, salary):
        self.name = name
        self.salary = salary

    def get_bonus(self):
        return self.salary * 0.1

    def __str__(self):
        return f"{self.name}: 薪资{self.salary}, 奖金{self.get_bonus():.2f}"

class Manager(Employee):
    def get_bonus(self):
        return self.salary * 0.2  # 经理奖金更高

class Developer(Employee):
    def __init__(self, name, salary, language):
        super().__init__(name, salary)
        self.language = language

    def __str__(self):
        return f"{super().__str__()}, 语言: {self.language}"

# 使用
employees = [
    Manager("张三", 10000),
    Developer("李四", 8000, "Python"),
    Employee("王五", 6000)
]

for emp in employees:
    print(emp)
# 张三: 薪资10000, 奖金2000.00
# 李四: 薪资8000, 奖金800.00, 语言: Python
# 王五: 薪资6000, 奖金600.00
```

### 2. 日志系统

```python
import logging

class Logger:
    def log(self, message):
        raise NotImplementedError

class FileLogger(Logger):
    def __init__(self, filename):
        self.filename = filename

    def log(self, message):
        with open(self.filename, "a") as f:
            f.write(f"{message}\n")

class ConsoleLogger(Logger):
    def log(self, message):
        print(f"[LOG] {message}")

def process_data(logger: Logger):
    logger.log("开始处理数据")
    # ... 处理逻辑
    logger.log("数据处理完成")

process_data(ConsoleLogger())
process_data(FileLogger("app.log"))
```

## 最佳实践

1. **优先组合而非继承**：避免过深的继承层次
2. **使用抽象基类**：定义清晰的接口
3. **遵循 LSP 原则**：子类应该能替换父类
4. **合理使用 super()**：确保正确调用父类方法
5. **保持单一职责**：每个类只做一件事

```python
# 组合优于继承的例子
class Engine:
    def start(self):
        return "引擎启动"

class Car:
    def __init__(self):
        self.engine = Engine()  # 组合

    def start(self):
        return self.engine.start()
```
