# 作用域与命名空间

[官方文档](https://docs.python.org/zh-cn/3/tutorial/classes.html#python-scopes-and-namespaces)

作用域决定了变量在程序中的可见性和生命周期。Python 使用 LEGB 规则来查找变量，理解作用域和命名空间对于避免变量冲突和编写清晰的代码至关重要。

## 命名空间概念

命名空间是从名称到对象的映射，类似于字典。

```python
# 内置命名空间
print(len([1, 2, 3]))  # len 在内置命名空间中

# 全局命名空间
x = 10  # x 在全局命名空间中

# 局部命名空间
def func():
    y = 20  # y 在局部命名空间中
    print(y)

func()
```

Python 中有三种主要的命名空间：
1. **内置命名空间**：包含内置函数和异常
2. **全局命名空间**：模块级别的变量
3. **局部命名空间**：函数内部的变量

## LEGB 规则

Python 按照以下顺序查找变量：

1. **L**ocal - 局部作用域
2. **E**nclosing - 嵌套函数作用域
3. **G**lobal - 全局作用域
4. **B**uilt-in - 内置作用域

```python
# 演示 LEGB 规则

x = "global"  # Global

def outer():
    x = "enclosing"  # Enclosing

    def inner():
        x = "local"  # Local
        print(x)

    inner()

outer()  # 输出: local

# 如果注释掉 local，会打印 enclosing
# 如果都注释掉，会打印 global
```

### 实际示例

```python
# Builtin
print(len([1, 2, 3]))  # 3（len 是内置函数）

# Global
message = "Hello"

def greet():
    # Enclosing
    prefix = "Greeting"

    def say_hello():
        # Local
        name = "Python"
        print(f"{prefix}: {message}, {name}!")

    say_hello()

greet()  # Greeting: Hello, Python!
```

## 局部作用域

函数内部定义的变量只能在函数内部访问。

```python
def my_func():
    local_var = "我是局部变量"
    print(local_var)

my_func()  # 我是局部变量
# print(local_var)  # NameError: name 'local_var' is not defined
```

### 参数也是局部变量

```python
def greet(name):
    # name 是局部变量
    message = f"Hello, {name}!"
    print(message)

greet("Python")
# print(name)  # NameError
```

## 全局作用域

模块级别定义的变量可以在整个模块中访问。

```python
global_var = "我是全局变量"

def read_global():
    # 可以读取全局变量
    print(global_var)

def write_global():
    # 这会创建新的局部变量，不会修改全局变量
    global_var = "新的局部变量"
    print(global_var)

read_global()   # 我是全局变量
write_global()  # 新的局部变量
print(global_var)  # 我是全局变量（未被修改）
```

### global 关键字

要在函数内部修改全局变量，需要使用 `global` 关键字。

```python
count = 0

def increment():
    global count
    count += 1

increment()
print(count)  # 1

increment()
print(count)  # 2
```

> 💡 **最佳实践**：尽量避免使用 `global`，优先使用参数传递和返回值。

## 嵌套作用域

嵌套函数可以访问外层函数的变量。

```python
def outer():
    outer_var = "外层变量"

    def inner():
        # 可以访问外层变量
        print(outer_var)

    inner()

outer()  # 外层变量
```

### nonlocal 关键字

要在嵌套函数中修改外层函数的变量，需要使用 `nonlocal`。

```python
def counter():
    count = 0

    def increment():
        nonlocal count
        count += 1
        return count

    return increment

# 创建计数器
counter1 = counter()
print(counter1())  # 1
print(counter1())  # 2
print(counter1())  # 3

# 独立的计数器
counter2 = counter()
print(counter2())  # 1
```

## 作用域示例

### 1. 闭包

```python
def make_multiplier(factor):
    """创建乘法器"""
    def multiply(number):
        return number * factor
    return multiply

double = make_multiplier(2)
triple = make_multiplier(3)

print(double(5))  # 10
print(triple(5))  # 15
```

### 2. 计数器

```python
def create_counter(start=0):
    """创建计数器"""
    count = start

    def counter():
        nonlocal count
        count += 1
        return count

    return counter

counter = create_counter(10)
print(counter())  # 11
print(counter())  # 12
```

### 3. 缓存装饰器

```python
def memoize(func):
    """简单的缓存装饰器"""
    cache = {}

    def wrapper(*args):
        if args not in cache:
            cache[args] = func(*args)
        return cache[args]

    return wrapper

@memoize
def fibonacci(n):
    if n < 2:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

print(fibonacci(50))  # 快速计算
```

## 常见错误

### 1. UnboundLocalError

```python
x = 10

def func():
    # print(x)  # UnboundLocalError!
    # Python 认为 x 是局部变量（因为后面有赋值）
    x = 20

func()
```

**解决**：使用 `global` 或改变变量名。

```python
x = 10

def func():
    global x
    print(x)  # 10
    x = 20

func()
print(x)  # 20
```

### 2. 循环变量泄漏（Python 2）

```python
# Python 2 中循环变量会泄漏到外部作用域
# for i in range(5):
#     pass
# print(i)  # 4（Python 2）

# Python 3 中已修复
for i in range(5):
    pass
# print(i)  # NameError（Python 3）
```

### 3. 可变对象的全局状态

```python
# 危险：全局可变状态
shared_list = []

def add_item(item):
    shared_list.append(item)

add_item(1)
add_item(2)
print(shared_list)  # [1, 2]

# 更好：使用类封装
class ItemCollector:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

collector = ItemCollector()
collector.add(1)
collector.add(2)
print(collector.items)  # [1, 2]
```

## globals() 和 locals()

```python
x = 10

def func():
    y = 20
    print("局部变量:", list(locals().keys()))
    print("全局变量:", list(globals().keys())[:5])

func()
# 局部变量: ['y']
# 全局变量: ['__name__', '__doc__', '__package__', ...]
```

## 模块作用域

每个模块有自己的全局作用域。

```python
# module_a.py
value = 10

# module_b.py
import module_a
print(module_a.value)  # 10
```

## 最佳实践

1. **最小化全局变量**：优先使用参数和返回值
2. **避免 shadowing**：不要用局部变量覆盖全局变量
3. **使用 nonlocal 谨慎**：优先考虑类或返回函数
4. **明确的命名**：避免名称冲突
5. **利用闭包**：封装状态和行为

```python
# 不好的实践
count = 0

def increment():
    global count
    count += 1

# 更好的实践
def make_counter():
    count = 0

    def increment():
        nonlocal count
        count += 1
        return count

    return increment

counter = make_counter()
```
