# 类型提示

[官方文档](https://docs.python.org/zh-cn/3/library/typing.html)

类型提示（Type Hints）是 Python 3.5+ 引入的功能，用于标注变量和函数的类型。虽然 Python 是动态类型语言，但类型提示可以提高代码可读性、帮助 IDE 提供智能提示，并支持静态类型检查。

## 基本语法

### 变量注解

```python
# 基本类型
name: str = "张三"
age: int = 25
height: float = 1.75
is_student: bool = True

# 容器类型
from typing import List, Dict, Set, Tuple

names: List[str] = ["张三", "李四"]
scores: Dict[str, int] = {"张三": 90}
unique_ids: Set[int] = {1, 2, 3}
point: Tuple[int, int] = (10, 20)
```

### 函数注解

```python
def greet(name: str) -> str:
    return f"Hello, {name}"

def add(a: int, b: int) -> int:
    return a + b

# 查看注解
print(greet.__annotations__)
# {'name': <class 'str'>, 'return': <class 'str'>}
```

## typing 模块

### 可选类型

```python
from typing import Optional

def get_name(user_id: int) -> Optional[str]:
    if user_id == 1:
        return "张三"
    return None
```

### 联合类型

```python
from typing import Union

def parse_value(value: Union[str, int]) -> str:
    return str(value)

# Python 3.10+ 可以使用 |
def parse_value(value: str | int) -> str:
    return str(value)
```

### 泛型

```python
from typing import TypeVar, Generic

T = TypeVar('T')

def first(items: list[T]) -> T:
    return items[0]

class Stack(Generic[T]):
    def __init__(self) -> None:
        self.items: list[T] = []

    def push(self, item: T) -> None:
        self.items.append(item)

    def pop(self) -> T:
        return self.items.pop()
```

## mypy 静态检查

### 安装和使用

```bash
pip install mypy
mypy script.py
```

### 示例

```python
# script.py
def add(a: int, b: int) -> int:
    return a + b

result = add("hello", "world")  # mypy 会报错
```

```bash
$ mypy script.py
script.py:4: error: Argument 1 to "add" has incompatible type "str"; expected "int"
```

## 最佳实践

1. **逐步添加**：从关键函数开始
2. **使用 mypy**：定期运行类型检查
3. **保持简单**：避免过度复杂的类型
4. **文档化**：类型提示不能替代文档字符串
5. **团队一致**：统一类型标注风格
