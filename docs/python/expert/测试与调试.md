# 测试与调试

[官方文档](https://docs.python.org/zh-cn/3/library/unittest.html)

测试和调试是保证代码质量的重要手段。Python 提供了 unittest 和 pytest 等测试框架，以及 pdb 调试工具。本文将介绍 Python 的测试和调试方法。

## unittest 框架

### 基本用法

```python
import unittest

def add(a, b):
    return a + b

class TestAdd(unittest.TestCase):
    def test_add_positive(self):
        self.assertEqual(add(2, 3), 5)

    def test_add_negative(self):
        self.assertEqual(add(-1, -1), -2)

    def test_add_zero(self):
        self.assertEqual(add(0, 0), 0)

if __name__ == "__main__":
    unittest.main()
```

### 常用断言

```python
class TestAssertions(unittest.TestCase):
    def test_assertions(self):
        self.assertEqual(1, 1)       # 相等
        self.assertNotEqual(1, 2)    # 不相等
        self.assertTrue(True)        # 为真
        self.assertFalse(False)      # 为假
        self.assertIsNone(None)      # 为 None
        self.assertIn(1, [1, 2, 3])  # 包含
        self.assertIsInstance(1, int) # 类型检查

    def test_exception(self):
        with self.assertRaises(ValueError):
            int("invalid")
```

### setUp 和 tearDown

```python
class TestDatabase(unittest.TestCase):
    def setUp(self):
        # 每个测试前执行
        self.conn = create_connection()

    def tearDown(self):
        # 每个测试后执行
        self.conn.close()

    def test_query(self):
        result = self.conn.query("SELECT 1")
        self.assertEqual(result, 1)
```

## pytest 简介

pytest 是更简洁的测试框架。

```bash
pip install pytest
```

```python
# test_sample.py
def add(a, b):
    return a + b

def test_add():
    assert add(2, 3) == 5
    assert add(-1, 1) == 0

def test_exception():
    with pytest.raises(ValueError):
        int("invalid")
```

```bash
pytest test_sample.py
```

## pdb 调试

### 基本命令

```python
import pdb

def buggy_function(x):
    pdb.set_trace()  # 设置断点
    y = x * 2
    z = y + 10
    return z
```

### 常用命令

| 命令 | 说明 |
| :--- | :--- |
| `n` | 执行下一行 |
| `s` | 进入函数 |
| `c` | 继续执行 |
| `p var` | 打印变量 |
| `l` | 显示代码 |
| `q` | 退出调试 |

### Python 3.7+ breakpoint()

```python
def buggy_function(x):
    breakpoint()  # 内置断点
    y = x * 2
    return y
```

## 最佳实践

1. **编写测试**：覆盖关键路径
2. **测试隔离**：每个测试独立
3. **命名规范**：test_ 开头
4. **使用 fixture**：复用测试数据
5. **持续集成**：自动运行测试
