# 性能优化

[官方文档](https://docs.python.org/zh-cn/3/library/profile.html)

性能优化是提升程序运行效率的关键。Python 提供了多种工具和技术来分析和优化代码性能。本文将介绍性能分析工具和优化技巧。

## 性能分析工具

### cProfile

```python
import cProfile

def slow_function():
    total = 0
    for i in range(1000000):
        total += i * i
    return total

# 分析性能
cProfile.run('slow_function()')
```

输出示例：
```

        1    0.000    0.000    0.000    0.000 {method 'disable' of '_lsprof.Profiler' objects}
```

### timeit

```python
import timeit

# 测试小段代码的执行时间
time = timeit.timeit('sum(range(1000))', number=10000)
print(f"执行 10000 次耗时: {time:.4f}s")

# 比较不同实现
def method1():
    return [x ** 2 for x in range(100)]

def method2():
    return list(map(lambda x: x ** 2, range(100)))

time1 = timeit.timeit(method1, number=10000)
time2 = timeit.timeit(method2, number=10000)
print(f"列表推导式: {time1:.4f}s")
print(f"map + lambda: {time2:.4f}s")
```

## 算法优化

### 选择合适的数据结构

```python
import time

# 列表查找 - O(n)
my_list = list(range(1000000))
start = time.time()
999999 in my_list
print(f"列表查找: {time.time() - start:.6f}s")

# 集合查找 - O(1)
my_set = set(range(1000000))
start = time.time()
999999 in my_set
print(f"集合查找: {time.time() - start:.6f}s")

# 字典查找 - O(1)
my_dict = {i: i for i in range(1000000)}
start = time.time()
999999 in my_dict
print(f"字典查找: {time.time() - start:.6f}s")
```

### 减少重复计算

```python
# 不好：重复计算
def process(items):
    for item in items:
        result = expensive_calculation(item)
        if expensive_calculation(item) > 10:  # 重复计算！
            print(result)

# 好：缓存结果
def process(items):
    for item in items:
        result = expensive_calculation(item)
        if result > 10:
            print(result)
```

## 数据结构选择

### 列表 vs 集合 vs 字典

```python
# 去重
# 不好 - O(n²)
unique = []
for item in items:
    if item not in unique:
        unique.append(item)

# 好 - O(n)
unique = list(set(items))

# 保持顺序的去重
seen = set()
unique = []
for item in items:
    if item not in seen:
        seen.add(item)
        unique.append(item)
```

### 生成器节省内存

```python
import sys

# 列表 - 占用大量内存
large_list = [x ** 2 for x in range(1000000)]
print(f"列表内存: {sys.getsizeof(large_list)} bytes")

# 生成器 - 内存友好
large_gen = (x ** 2 for x in range(1000000))
print(f"生成器内存: {sys.getsizeof(large_gen)} bytes")
```

## C 扩展简介

### 使用 numpy

```python
import numpy as np
import time

# Python 列表
start = time.time()
result = sum([i * i for i in range(10000000)])
print(f"Python 列表: {time.time() - start:.4f}s")

# NumPy 数组
start = time.time()
arr = np.arange(10000000)
result = np.sum(arr * arr)
print(f"NumPy: {time.time() - start:.4f}s")
# NumPy 快 10-100 倍
```

# cython_example.pyx
def sum_of_squares(int n):

## 常见优化技巧

### 1. 局部变量更快

```python
# 慢
def slow():
    for i in range(1000000):
        math.sqrt(i)

# 快
def fast():
    sqrt = math.sqrt  # 局部变量
    for i in range(1000000):
        sqrt(i)
```

### 2. 字符串拼接

```python
# 慢 - O(n²)
result = ""
for s in strings:
    result += s

# 快 - O(n)
result = "".join(strings)
```

### 3. 列表推导式优于循环

```python
# 慢
squares = []
for x in range(1000):
    squares.append(x ** 2)

# 快
squares = [x ** 2 for x in range(1000)]
```

### 4. 使用内置函数

```python
# 慢
max_value = data[0]
for item in data:
    if item > max_value:
        max_value = item

# 快
max_value = max(data)
```

## 最佳实践

1. **先测量再优化**：使用 cProfile 找出瓶颈
2. **选择合适的算法**：O(n) 优于 O(n²)
3. **利用内置函数**：通常用 C 实现，更快
4. **减少内存分配**：复用对象，避免频繁创建
5. **考虑 C 扩展**：关键路径使用 numpy/Cython
