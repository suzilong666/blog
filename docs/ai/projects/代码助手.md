# 代码助手开发实战

## 项目概述

开发一个智能代码助手，能够提供代码补全、解释、审查和重构建议等功能。

### 核心功能

- ✅ 代码解释：理解并解释代码功能
- ✅ 代码审查：发现潜在问题和改进点
- ✅ 代码生成：根据描述生成代码
- ✅ Bug 修复：定位并修复错误
- ✅ 代码优化：性能和质量优化建议

## 技术栈

- **后端**: FastAPI
- **AI**: OpenAI GPT-4 / GPT-3.5-turbo
- **前端**: VS Code Extension 或 Web 界面

## 实现方案

### 1. 核心引擎

```python
"""
代码助手核心引擎
"""
from openai import OpenAI
from typing import Optional

class CodeAssistant:
    """智能代码助手"""
    
    def __init__(self, model: str = "gpt-4"):
        self.client = OpenAI()
        self.model = model
    
    def explain_code(self, code: str, language: str = "python") -> str:
        """解释代码"""
        prompt = f"""请详细解释以下{language}代码的功能：

```{language}
{code}
```

请按以下结构回答：
1. 整体功能概述
2. 关键逻辑说明
3. 输入输出说明
4. 使用示例"""

        response = self.client.chat.completions.create(
            model=self.model,
            messages=[{"role": "user", "content": prompt}],
            temperature=0.3
        )
        
        return response.choices[0].message.content
    
    def review_code(self, code: str, language: str = "python") -> dict:
        """代码审查"""
        prompt = f"""请审查以下{language}代码，从以下角度分析：

```{language}
{code}
```

审查维度：
1. 潜在 Bug
2. 性能问题
3. 代码规范
4. 安全性
5. 可维护性

请以 JSON 格式输出，包含字段：issues（问题列表）、suggestions（建议）、score（评分1-10）"""

        response = self.client.chat.completions.create(
            model=self.model,
            messages=[{"role": "user", "content": prompt}],
            temperature=0.2
        )
        
        import json
        try:
            return json.loads(response.choices[0].message.content)
        except:
            return {"error": "解析失败"}
    
    def generate_code(self, description: str, language: str = "python") -> str:
        """生成代码"""
        prompt = f"""根据以下需求生成{language}代码：

需求：{description}

要求：
- 代码完整可运行
- 包含必要的注释
- 遵循最佳实践
- 处理边界情况

只输出代码，不要解释。"""

        response = self.client.chat.completions.create(
            model=self.model,
            messages=[{"role": "user", "content": prompt}],
            temperature=0.3
        )
        
        return response.choices[0].message.content
    
    def fix_bug(self, code: str, error_message: str) -> str:
        """修复 Bug"""
        prompt = f"""以下代码出现错误，请修复：

代码：
```python
{code}
```

错误信息：
{error_message}

请：
1. 分析问题原因
2. 提供修复后的完整代码
3. 说明修复内容"""

        response = self.client.chat.completions.create(
            model=self.model,
            messages=[{"role": "user", "content": prompt}]
        )
        
        return response.choices[0].message.content
    
    def optimize_code(self, code: str) -> str:
        """优化代码"""
        prompt = f"""请优化以下代码的性能和可读性：

```python
{code}
```

要求：
1. 保持功能不变
2. 提升执行效率
3. 改善代码结构
4. 说明优化点

提供优化后的完整代码。"""

        response = self.client.chat.completions.create(
            model=self.model,
            messages=[{"role": "user", "content": prompt}]
        )
        
        return response.choices[0].message.content
```

### 2. API 服务

```python
"""
FastAPI 服务
"""
from fastapi import FastAPI
from pydantic import BaseModel
from typing import Optional

app = FastAPI(title="代码助手 API")
assistant = CodeAssistant()

class CodeRequest(BaseModel):
    code: str
    language: Optional[str] = "python"

class GenerateRequest(BaseModel):
    description: str
    language: Optional[str] = "python"

class FixRequest(BaseModel):
    code: str
    error_message: str

@app.post("/api/explain")
def explain(request: CodeRequest):
    """解释代码"""
    result = assistant.explain_code(request.code, request.language)
    return {"explanation": result}

@app.post("/api/review")
def review(request: CodeRequest):
    """代码审查"""
    result = assistant.review_code(request.code, request.language)
    return result

@app.post("/api/generate")
def generate(request: GenerateRequest):
    """生成代码"""
    result = assistant.generate_code(request.description, request.language)
    return {"code": result}

@app.post("/api/fix")
def fix_bug(request: FixRequest):
    """修复 Bug"""
    result = assistant.fix_bug(request.code, request.error_message)
    return {"fixed_code": result}

@app.post("/api/optimize")
def optimize(request: CodeRequest):
    """优化代码"""
    result = assistant.optimize_code(request.code)
    return {"optimized_code": result}
```

```javascript
const vscode = require('vscode');
const axios = require('axios');

const API_URL = 'http://localhost:8000/api';

function activate(context) {
    // 解释代码命令
    let explainCommand = vscode.commands.registerCommand(
        'aiCodeAssistant.explain',
        async () => {
            const editor = vscode.window.activeTextEditor;
            if (!editor) return;
            
            const selectedCode = editor.document.getText(editor.selection);
            
            if (!selectedCode) {
                vscode.window.showErrorMessage('请先选择代码');
                return;
            }
            
            vscode.window.withProgress(
                { location: vscode.ProgressLocation.Notification },
                async (progress) => {
                    progress.report({ message: "AI 正在分析代码..." });
                    
                    try {
                        const response = await axios.post(`${API_URL}/explain`, {
                            code: selectedCode,
                            language: editor.document.languageId
                        });
                        
                        // 在侧边栏显示结果
                        showExplanation(response.data.explanation);
                        
                    } catch (error) {
                        vscode.window.showErrorMessage('请求失败: ' + error.message);
                    }
                }
            );
        }
    );
    
    context.subscriptions.push(explainCommand);
}

function showExplanation(text) {
    const panel = vscode.window.createWebviewPanel(
        'codeExplanation',
        '代码解释',
        vscode.ViewColumn.Beside
    );
    
    panel.webview.html = `
        <!DOCTYPE html>
        <html>
        <head>
            <style>
                body { 
                    font-family: Arial, sans-serif; 
                    padding: 20px;
                    line-height: 1.6;
                }
                pre {
                    background: #f5f5f5;
                    padding: 10px;
                    border-radius: 5px;
                    overflow-x: auto;
                }
            </style>
        </head>
        <body>
            ${text.replace(/\n/g, '<br>')}
        </body>
        </html>
    `;
}

exports.activate = activate;
```

## 使用示例

### 1. 代码解释

**输入：**
```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```

**AI 输出：**
```
整体功能：
这是一个计算斐波那契数列的递归函数。

关键逻辑：
1. 基准情况：当 n <= 1 时，直接返回 n
2. 递归情况：返回前两个数的和

时间复杂度：O(2^n)，存在大量重复计算
空间复杂度：O(n)，递归调用栈深度

改进建议：
可以使用记忆化或动态规划优化到 O(n) 时间复杂度。
```

```python
def stream_code_review(code: str):
    """流式返回代码审查结果"""
    
    stream = client.chat.completions.create(
        model="gpt-4",
        messages=[{"role": "user", "content": f"审查代码：{code}"}],
        stream=True
    )
    
    for chunk in stream:
        if chunk.choices[0].delta.content:
            yield chunk.choices[0].delta.content
```

    if code_pattern in pattern_cache:
