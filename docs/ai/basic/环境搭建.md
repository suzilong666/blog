## API 密钥获取与管理

### OpenAI API Key

#### 1. 注册账号

访问 [OpenAI Platform](https://platform.openai.com/) 注册账号。

#### 2. 创建 API Key

1. 登录后进入 [API Keys 页面](https://platform.openai.com/api-keys)
2. 点击 "Create new secret key"
3. 为 Key 命名（如 "dev-key"）
4. 复制并安全保存 Key（只显示一次！）

```
sk-proj-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

#### 3. 设置用量限制

在 [Usage limits](https://platform.openai.com/account/limits) 设置月度预算上限，避免意外费用。

#### 4. 安全存储

**❌ 不要这样做：**
```python
# 硬编码在代码中 - 危险！
api_key = "sk-proj-xxxxx"
```

**✅ 推荐做法：**

使用环境变量：
```bash
# .env 文件
OPENAI_API_KEY=sk-proj-xxxxx
```

```python
# 代码中读取
import os
from dotenv import load_dotenv

load_dotenv()
api_key = os.getenv("OPENAI_API_KEY")
```

`hello_openai.py`：

```python
"""
OpenAI API Hello World 示例
"""
import os
from openai import OpenAI
from dotenv import load_dotenv

# 加载环境变量
load_dotenv()

# 初始化客户端
client = OpenAI(
    api_key=os.getenv("OPENAI_API_KEY")
)

def chat_with_gpt(prompt: str) -> str:
    """
    与 GPT 进行对话
    
    Args:
        prompt: 用户输入的问题
        
    Returns:
        AI 的回答
    """
    try:
        response = client.chat.completions.create(
            model="gpt-3.5-turbo",
            messages=[
                {"role": "system", "content": "你是一个友好的AI助手。"},
                {"role": "user", "content": prompt}
            ],
            temperature=0.7,
            max_tokens=500
        )
        
        return response.choices[0].message.content
    
    except Exception as e:
        print(f"Error: {e}")
        return None

if __name__ == "__main__":
    print("🤖 AI 助手已启动！（输入 'quit' 退出）\n")
    
    while True:
        user_input = input("👤 你: ")
        
        if user_input.lower() == 'quit':
            print("👋 再见！")
            break
        
        if not user_input.strip():
            continue
        
        print("🤖 AI: ", end="", flush=True)
        response = chat_with_gpt(user_input)
        
        if response:
            print(response)
            print()  # 空行分隔
        else:
            print("抱歉，我遇到了一些问题。\n")
```

#### 3. 代码实现

`hello-openai.js`：
```javascript
/**
 * OpenAI API Hello World 示例
 */
import OpenAI from 'openai';
import dotenv from 'dotenv';

// 加载环境变量
dotenv.config();

// 初始化客户端
const openai = new OpenAI({
  apiKey: process.env.OPENAI_API_KEY
});

/**
 * 与 GPT 进行对话
 * @param {string} prompt - 用户输入的问题
 * @returns {Promise<string>} AI 的回答
 */
async function chatWithGPT(prompt) {
  try {
    const response = await openai.chat.completions.create({
      model: "gpt-3.5-turbo",
      messages: [
        { role: "system", content: "你是一个友好的AI助手。" },
        { role: "user", content: prompt }
      ],
      temperature: 0.7,
      max_tokens: 500
    });
    
    return response.choices[0].message.content;
  } catch (error) {
    console.error('Error:', error.message);
    return null;
  }
}

// 主程序
async function main() {
  const readline = await import('readline');
  
  const rl = readline.createInterface({
    input: process.stdin,
    output: process.stdout
  });
  
  const question = (query) => new Promise((resolve) => rl.question(query, resolve));
  
  console.log('🤖 AI 助手已启动！（输入 \'quit\' 退出）\n');
  
  while (true) {
    const userInput = await question('👤 你: ');
    
    if (userInput.toLowerCase() === 'quit') {
      console.log('👋 再见！');
      rl.close();
      break;
    }
    
    if (!userInput.trim()) {
      continue;
    }
    
    process.stdout.write('🤖 AI: ');
    const response = await chatWithGPT(userInput);
    
    if (response) {
      console.log(response + '\n');
    } else {
      console.log('抱歉，我遇到了一些问题。\n');
    }
  }
}

main();
```

## 常见问题

### Q1: 遇到 "Invalid API Key" 错误

**原因：**
- API Key 复制不完整
- 环境变量未正确加载
- Key 已被撤销

**解决：**
```python
# 检查 API Key 是否正确加载
import os
print(os.getenv("OPENAI_API_KEY"))
# 应该输出完整的 Key，而不是 None
```

### Q2: 遇到 "Rate limit exceeded" 错误

**原因：**
- 请求频率超过限制
- 账户额度不足

**解决：**
```python
import time

def safe_api_call(func, *args, **kwargs):
    """安全的 API 调用，包含重试机制"""
    max_retries = 3
    for attempt in range(max_retries):
        try:
            return func(*args, **kwargs)
        except RateLimitError:
            if attempt < max_retries - 1:
                wait_time = 2 ** attempt  # 指数退避
                print(f"速率限制，等待 {wait_time} 秒后重试...")
                time.sleep(wait_time)
            else:
                raise
```

### Q3: 响应速度很慢

**优化建议：**
1. 使用流式响应（Streaming）
2. 减少 max_tokens
3. 选择更快的模型（如 gpt-3.5-turbo）
4. 缩短 prompt 长度

### Q4: 如何控制成本？

**策略：**
1. 设置月度预算上限
2. 使用更便宜的模型进行测试
3. 缓存常见问题的回答
4. 优化 prompt，减少不必要的 token

```python
# 估算 token 成本
def estimate_cost(input_text: str, output_tokens: int, model: str = "gpt-3.5-turbo") -> float:
    """估算 API 调用成本"""
    # 粗略估算：1 token ≈ 4 个字符
    input_tokens = len(input_text) / 4
    
    # GPT-3.5-turbo 价格（每 1K tokens）
    prices = {
        "gpt-3.5-turbo": {"input": 0.0005, "output": 0.0015},
        "gpt-4": {"input": 0.03, "output": 0.06},
    }
    
    price = prices.get(model, prices["gpt-3.5-turbo"])
    cost = (input_tokens / 1000 * price["input"]) + (output_tokens / 1000 * price["output"])
    
    return round(cost, 6)

# 使用示例
cost = estimate_cost("你好，请介绍一下自己", 100)
print(f"预计成本: ${cost}")
```
