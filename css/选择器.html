选择器 ​

CSS 选择器规定了 CSS 规则会被应用到哪些元素上。

基本选择器 ​

通用选择器 ​

css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

元素选择器 ​

css
p {
  color: #333;
  line-height: 1.6;
}

类选择器 ​

css
.button {
  padding: 10px 20px;
  border: none;
  border-radius: 4px;
}

ID选择器 ​

css
#header {
  background-color: #f8f9fa;
  padding: 20px;
}

属性选择器 ​

css
/* 基础属性选择器 */
[disabled] {
  opacity: 0.5;
  cursor: not-allowed;
}

/* 精确匹配 */
[type="text"] {
  border: 1px solid #ccc;
}

[href="https://example.com"] {
  color: green;
}

/* 开始匹配 ^= */
a[href^="https"] {
  /* 所有https链接 */
  color: green;
}

[class^="icon-"] {
  /* 所有以icon-开头的类 */
  background-image: url("icons.png");
}

/* 结束匹配 $= */
a[href$=".pdf"]::after {
  content: " 📄";
  font-size: 0.8em;
}

[src$=".jpg"] {
  border: 2px solid #ddd;
}

/* 包含匹配 *= */
[class*="button"] {
  /* 类名包含button */
  cursor: pointer;
}

[href*="google"] {
  /* href包含google */
  color: #4285f4;
}

/* 单词匹配 ~= */
[class~="active"] {
  /* 类名中有独立的active单词 */
  font-weight: bold;
}

/* 连字符匹配 |= */
[lang|="en"] {
  /* lang属性以en开头或en-开头 */
  font-family: "Arial", sans-serif;
}

分组选择器（Grouping selector） ​

选择器列表 ​

, 是将不同的选择器组合在一起的方法，它选择所有能被列表中的任意一个选择器选中的节点。

css
p, a {
  color: red;
}

组合器（Combinator） ​

后代组合器 ​

“ ”（空格）组合器选择前一个元素的后代节点。

语法：A B

css
/* div span 匹配所有位于任意 <div> 元素之内的 <span> 元素。 */
div span {

}

直接子代组合器 ​

> 组合器选择前一个元素的直接子代的节点。

语法：A > B

css
/* ul > li 匹配直接嵌套在 <ul> 元素内的所有 <li> 元素。*/
ul > li {

}

一般兄弟组合器 ​

~ 组合器选择兄弟元素，也就是说，后一个节点在前一个节点后面的任意位置，并且共享同一个父节点。

语法：A ~ B

css
/* p ~ span 匹配同一父元素下，<p> 元素后的所有 <span> 元素。 */
p ~ span{

}

紧邻兄弟组合器 ​

+ 组合器选择相邻元素，即后一个元素紧跟在前一个之后，并且共享同一个父节点。

语法：A + B

css
/* h2 + p 会匹配紧邻在 h2 元素后的第一个 <p> 元素。 */
h2 + p {

}

列组合器(实验性) ​

|| 组合器选择属于某个表格行的节点。

语法：A || B

css
/* col || td 会匹配所有 <col> 作用域内的 <td> 元素。 */
col || td {

}

伪选择器（Pseudo） ​

伪类 ​

: 伪选择器支持按照未被包含在文档树中的状态信息来选择元素。

css
/* a:visited 匹配所有曾被访问过的 <a> 元素。*/
a:visited {

}

伪元素 ​

:: 伪选择器用于表示无法用 HTML 语义表达的实体。

css
/* p::first-line 匹配所有 <p> 元素的第一行。*/
p::first-line {
    
}
