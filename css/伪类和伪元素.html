伪类（Pseudo-classes） ​

作用：为元素的特定状态添加样式

语法：单个冒号 :（CSS3规范也支持 ::，但推荐用 :）

特点：选择处于特定状态的元素

常用伪类分类 ​

1. 链接相关 ​

css
/* 未访问的链接 */
a:link { color: blue; }

/* 已访问的链接 */
a:visited { color: purple; }

/* 鼠标悬停 */
a:hover { 
    color: red;
    text-decoration: underline;
}

/* 被激活/点击时 */
a:active { color: orange; }

/* 获得焦点时 */
a:focus { outline: 2px solid #4CAF50; }

2. 表单状态 ​

css
/* 启用状态 */
input:enabled { border-color: #ccc; }

/* 禁用状态 */
input:disabled { 
    opacity: 0.5;
    cursor: not-allowed;
}

/* 选中状态 */
input:checked {
    accent-color: #4CAF50;
}

/* 必填项 */
input:required { border-left: 3px solid red; }

/* 有效输入 */
input:valid { border-color: green; }

/* 无效输入 */
input:invalid { border-color: red; }

/* 范围输入 */
input:in-range { background: #e8f5e8; }
input:out-of-range { background: #ffe6e6; }

3. 结构选择 ​

css
/* 第一个子元素 */
ul li:first-child { font-weight: bold; }

/* 最后一个子元素 */
ul li:last-child { margin-bottom: 0; }

/* 第n个子元素 */
ul li:nth-child(3) { color: red; }
ul li:nth-child(odd) { background: #f5f5f5; }  /* 奇数 */
ul li:nth-child(even) { background: #eaeaea; } /* 偶数 */
ul li:nth-child(3n) { background: lightblue; } /* 3的倍数 */
ul li:nth-child(3n+1) { background: lightgreen; }

/* 倒数第n个子元素 */
ul li:nth-last-child(2) { color: blue; }

/* 唯一子元素 */
div:only-child { border: 2px solid green; }

/* 特定类型的第一个元素 */
p:first-of-type { font-size: 1.2em; }

/* 没有子元素的元素 */
div:empty { display: none; }

/* 排除某些元素 */
.container > *:not(.exclude) { 
    margin-bottom: 10px; 
}

/* 目标元素（URL锚点） */
:target {
    background-color: yellow;
    animation: highlight 1s ease;
}

4. 用户交互 ​

css
/* 获取焦点的元素 */
input:focus {
    box-shadow: 0 0 5px rgba(81, 203, 238, 1);
}

/* 焦点内的元素（可用于构建焦点环） */
.element:focus-within {
    border: 2px solid #4CAF50;
}

/* 用户可选择的元素 */
::selection {
    background-color: #4CAF50;
    color: white;
}

/* 拖放的元素 */
.draggable:active {
    cursor: grabbing;
}

伪元素（Pseudo-elements） ​

伪元素不是DOM的一部分：无法通过JavaScript直接操作

css
/* ::before - 在元素内容前插入 */
.button::before {
    content: "🔗 ";
    margin-right: 5px;
}

/* ::after - 在元素内容后插入 */
.tooltip::after {
    content: attr(data-tooltip);
    position: absolute;
    background: black;
    color: white;
    padding: 5px;
    border-radius: 3px;
    white-space: nowrap;
}

/* ::first-letter - 首字母样式 */
p::first-letter {
    font-size: 200%;
    color: #8A2BE2;
    float: left;
    line-height: 1;
    margin-right: 5px;
}

/* ::first-line - 首行样式 */
p::first-line {
    font-weight: bold;
    color: #333;
}

/* ::placeholder - 输入框占位符 */
input::placeholder {
    color: #999;
    font-style: italic;
}

/* ::selection - 选中文本样式 */
::selection {
    background: #FF4081;
    color: white;
}

/* ::marker - 列表标记样式 */
li::marker {
    color: #4CAF50;
    content: "✓ ";
}

/* ::backdrop - 全屏/对话框背景 */
dialog::backdrop {
    background: rgba(0, 0, 0, 0.5);
}
